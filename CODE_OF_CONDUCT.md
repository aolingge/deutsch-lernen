# Code of Conduct

这个仓库欢迎围绕公开德语学习资料的贡献、勘误和整理建议。为了让讨论对中文学习者、德语学习者和外部贡献者都可用，请遵守下面的基本规则。

## Our Standards

- 尊重他人，讨论内容本身，不攻击个人背景、语言水平、国籍或身份。
- 对初学者友好。指出问题时尽量说明原因，并给出更清楚的改法。
- 优先引用公开、官方、可核查的资料，不贴私人聊天、课堂截图、受版权保护的大段原文或个人申请材料。
- 不分享真实姓名、联系方式、住址、证件信息、账号、订单号、私人链接或凭据。
- 不发布仇恨、骚扰、歧视、威胁、色情或明显违法内容。

## Scope

本准则适用于仓库中的 issue、pull request、讨论文本、示例内容和维护者沟通。

## Reporting

如果你看到明显不当行为、敏感信息泄露或不适合公开仓库的内容：

1. 先不要复制和扩散敏感内容。
2. 按 [SECURITY.md](SECURITY.md) 的方式报告私密数据、凭据或版权问题。
3. 普通协作问题可以直接开 issue 或 pull request 说明。

## Enforcement

维护者可以编辑、隐藏、关闭或删除不符合本准则的内容；必要时可限制相关账号继续互动。
