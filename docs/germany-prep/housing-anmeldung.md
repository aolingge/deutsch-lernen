# 住房与 Anmeldung

`Anmeldung = 住址登记 / 抵德后向城市登记当前住址 / 不同城市预约和材料要求不同。`

## 官方入口

- [DAAD accommodation finder](https://www.daad.de/en/studying-in-germany/living-in-germany/accommodation-finder/)
- [DSW accommodation](https://www.internationale-studierende.de/en/prepare-your-studies/accommodation)
- [Berlin Anmeldung](https://willkommenszentrum.berlin.de/wohnen/anmeldung-fuer-die-wohnung)
- [Munich residence registration](https://stadt.muenchen.de/infos/residence-registration.html)

## 常见材料词

| Deutsch | 中文 |
| --- | --- |
| der Mietvertrag | 租房合同 |
| die Miete | 房租 |
| die Kaution | 押金 |
| die Nebenkosten | 附加费用 |
| die Wohnungsgeberbestaetigung | 房东入住确认 |
| der Termin | 预约 |

## 场景表达

```text
Ist die Wohnung noch frei?
Wann kann ich die Wohnung besichtigen?
Koennten Sie mir die Wohnungsgeberbestaetigung ausstellen?
Ich moechte mich anmelden.
Welche Unterlagen brauche ich?
```

## 安全边界

- 不公开个人住址、合同、房东信息、预约截图。
- 不传播假地址 Anmeldung。
- 不提前向不可核验对象支付大额押金。
