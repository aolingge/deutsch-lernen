# B1 口语模板

B1 口语先追求“能稳定说完整”，再追求复杂表达。句子短一点没关系，逻辑清楚更重要。

## 常用观点句

```text
Meiner Meinung nach ...
Ich denke, dass ...
Ich finde, dass ...
Ich bin der Meinung, dass ...
Aus meiner Sicht ...
```

## 给理由

```text
Der Grund ist, dass ...
weil ...
denn ...
Deshalb ...
Aus diesem Grund ...
```

## 举例

```text
Zum Beispiel ...
Ein gutes Beispiel ist ...
Beispielsweise ...
```

## 对比

```text
Einerseits ..., andererseits ...
Zwar ..., aber ...
Trotzdem ...
```

## 同意和反对

同意：

```text
Da stimme ich dir zu.
Genau.
Das finde ich auch.
Das ist eine gute Idee.
```

委婉反对：

```text
Ich sehe das anders.
Da bin ich nicht ganz deiner Meinung.
Das klingt gut, aber ...
Ich bin damit nicht einverstanden, weil ...
```

## Teil 1：计划协商

常见任务：

- 生日派对或朋友聚会
- 周末旅行
- 德语学习小组
- 共同观影或看演出
- 徒步、运动活动

可以围绕四个问题推进：

| 问题 | 德语表达 |
| --- | --- |
| 去哪里 | Wohin koennten wir gehen/fahren? |
| 什么时候 | Wann wollen wir uns treffen? |
| 怎么去 | Wie kommen wir dorthin? |
| 准备什么 | Was muessen wir mitbringen? |

常用互动句：

```text
Hast du schon eine Idee?
Wie waere es mit ...?
Haettest du Lust auf ...?
Koennten wir ...?
Waere es moeglich, ...?
Was meinst du dazu?
Dann entscheiden wir uns fuer ...
Also machen wir das so.
```

## Teil 2：小演讲

简单结构：

```text
Das Thema meiner Praesentation ist ...

Ich moechte zuerst ueber meine persoenliche Erfahrung sprechen.
Danach spreche ich ueber die Situation in meinem Land.
Zum Schluss nenne ich Vorteile und Nachteile und sage meine Meinung.

Meiner Meinung nach ...
Ein Vorteil ist, dass ...
Ein Nachteil ist, dass ...
Zum Schluss finde ich, dass ...
```

## Teil 3：问答互动

提问：

```text
Warum findest du das wichtig?
Welche Vorteile siehst du?
Gibt es auch Nachteile?
Wie ist die Situation in deinem Land?
Was wuerdest du empfehlen?
```

回答：

```text
Das ist eine interessante Frage.
Ich denke, dass ...
Ein Beispiel dafuer ist ...
Ich wuerde empfehlen, dass ...
```

没听清时：

```text
Kannst du die Frage bitte wiederholen?
Kannst du das bitte einfacher sagen?
Entschuldigung, ich habe dich nicht ganz verstanden.
```
