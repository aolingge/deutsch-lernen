# 德语考试地图：A1-C1、TestDaF、DSH

这页帮助你判断该准备哪个考试。考试事实会变化，报名、费用、考位和承认规则必须回官方页面或目标大学页面确认。

Last checked: 2026-05-05

## 先看用途

| 目标 | 常见考试 | 说明 |
| --- | --- | --- |
| 证明基础德语 | Goethe/telc/OeSD A1-A2 | 入门能力证明，常见于基础签证或课程场景 |
| 生活独立和融入 | Goethe B1、telc B1、DTZ | 常见生活、融入课程或日常沟通目标 |
| 工作和职业发展 | Goethe/telc B2、telc Beruf | 很多工作场景更看重 B2 以上沟通 |
| 德语授课大学 | TestDaF、DSH、telc Deutsch C1 Hochschule、Goethe C2 或学校认可考试 | 具体承认规则以目标大学为准 |
| 学术德语能力 | DSH、TestDaF、telc C1 Hochschule | 重点是学术阅读、听讲座、写作和讨论 |

## A1-C1 与考试准备

| 等级 | 可以优先准备 | 训练重点 |
| --- | --- | --- |
| A1 | Goethe Start Deutsch 1 / telc A1 | 自我介绍、问答、数字时间、简单邮件 |
| A2 | Goethe A2 / telc A2 | 日常经历、预约、购物、交通、简单过去 |
| B1 | Goethe B1 / telc B1 / DTZ | 四项任务、观点理由、邮件和口语互动 |
| B2 | Goethe B2 / telc B2 | 新闻、抽象话题、较长写作和讨论 |
| C1 | Goethe C1 / telc C1 Hochschule / TestDaF / DSH | 学术阅读、讲座听力、摘要、论证、报告 |

## 高校德语证明要点

- HRK 说明，符合框架的 DSH、TestDaF、Studienkolleg 德语部分、DSD II 等可作为德国高校德语学习能力证明。
- 很多大学常见要求是 TestDaF 各项 TDN 4 或 DSH-2，但具体专业可能有不同要求。
- telc Deutsch C1 Hochschule 是高校入学方向的 C1 考试，是否接受仍应查目标大学招生页面。
- 不要只看论坛经验，最终以目标大学、uni-assist、HRK/考试机构官方说明为准。

## 选择建议

| 情况 | 建议 |
| --- | --- |
| 你还在 A1-B1 | 先不要纠结 DSH/TestDaF，稳住听说读写基本功 |
| 你目标是德国生活 | B1-B2 和生活场景德语优先 |
| 你目标是德国大学 | B2 后开始比较 TestDaF、DSH、telc C1 Hochschule |
| 你有明确目标大学 | 先读该大学语言证明页面，再决定考试 |
| 你不确定专业是否德语授课 | 先查课程页面的 language of instruction |

## 风险边界

- 不上传官方 PDF、音频、题册全文。
- 不传播“内部题库”“保过”“买证”。
- 不把个人成绩单、准考证、报名截图放进仓库。
- 不把某个论坛经验当成通用政策。
