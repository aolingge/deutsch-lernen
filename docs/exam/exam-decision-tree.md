# 德语考试选择树

考试选择先看用途，再看等级。不要先问“哪个考试简单”，要先问“我的目标机构承认哪个考试”。

## 快速结论

| 目标 | 优先考虑 |
| --- | --- |
| 入门证明 | Goethe/telc/OeSD A1-A2 |
| 生活独立或融入 | Goethe B1、telc B1、DTZ |
| 工作沟通 | B2 或职业德语方向考试 |
| 德语授课大学 | TestDaF、DSH、telc Deutsch C1 Hochschule 或目标大学认可考试 |
| 学术德语训练 | DSH/TestDaF/C1 路线 |

官方入口：

- [Goethe exams](https://www.goethe.de/en/spr/prf.html)
- [telc exams](https://www.telc.net/sprachpruefungen/)
- [TestDaF](https://www.testdaf.de/de/)
- [FaDaF DSH](https://www.fadaf.de/dsh/start/was-ist-die-dsh/)
- [HRK Sprachnachweis Deutsch](https://www.hrk.de/themen/internationales/internationale-studierende-und-forschende/hochschulzugang-fuer-internationale-studierende/sprachnachweis-deutsch/)
- [DAAD German language](https://www.daad.de/en/study-and-research-in-germany/plan-your-studies/the-german-language/)

## 第一步：你为什么考试

```text
只是证明基础能力 -> A1/A2
想生活独立 -> B1/B2
想申请大学 -> 先查目标大学语言证明页面
想进德语授课专业 -> B2-C1 后再定 TestDaF/DSH/telc C1 Hochschule
```

如果目标是德国大学，不要只看论坛经验。先查：

1. 目标大学专业页面。
2. 目标大学国际学生/语言要求页面。
3. HRK、DAAD、考试机构官方说明。

## 第二步：按等级接训练

| 当前等级 | 不建议 | 建议 |
| --- | --- | --- |
| A1-A2 | 纠结 DSH/TestDaF | 先用 [A1-C1 总路线](../learning-paths/a1-c1-roadmap.md) 建基础 |
| B1 | 直接刷 C1 真题 | 用 [B1 路线](../learning-paths/b1-roadmap.md) 做四项闭环 |
| B2 | 只背模板 | 进入 [B2 自检](../levels/b2-self-check.md) 和长材料训练 |
| C1 | 只泛读新闻 | 按 [DSH 模块练习](dsh-module-drills.md) 或 [TestDaF 衔接](testdaf-c1-bridge.md) 做模块训练 |

## TestDaF / DSH / telc C1 Hochschule 怎么选

| 考试 | 更适合谁 | 注意 |
| --- | --- | --- |
| TestDaF | 需要标准化考试、想跨城市/跨学校提交成绩的人 | 目标大学常见要求是各项达到指定 TDN，必须查学校 |
| DSH | 已在某大学语言中心或目标大学接受 DSH 的人 | 考点和样题风格由注册高校执行，必须查目标考点 |
| telc C1 Hochschule | 想用 C1 Hochschule 方向证明大学语言能力的人 | 是否接受仍以目标大学为准 |

## 风险边界

- 不上传官方题册、PDF、音频或答案。
- 不传播来源不明真题包。
- 不把报名截图、成绩单、准考证、证件信息放进仓库。
- 考试政策、费用、考位、承认规则会变化，最终以官方页面和目标机构为准。
