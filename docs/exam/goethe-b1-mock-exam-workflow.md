# Goethe B1 官方模考使用路线

官方模考不要反复刷分。第一次熟悉题型，第二次限时，第三次只复盘错因。

## 使用顺序

1. Goethe 在线 Modellsatz：低压力熟悉题型。
2. Goethe PDF + 音频：按真实时间做模块。
3. telc/OESD B1：作为补充迁移检测。
4. 回到 Goethe：考前只用 Goethe 规则和评分标准校准。

## 三轮使用法

| 轮次 | 目的 | 动作 |
| --- | --- | --- |
| 第 1 轮 | 熟悉题型 | 不计分，知道每个 Teil 要做什么 |
| 第 2 轮 | 限时训练 | 严格按时间做，记录分数 |
| 第 3 轮 | 错因复盘 | 只看错题、作文、口语录音 |

## 记录命名

```text
2026-05-05-Goethe-B1-Lesen-Modellsatz.md
2026-05-05-Goethe-B1-Hoeren-Teil2-audio-note.md
2026-05-05-Goethe-B1-Schreiben-Teil1-draft.md
2026-05-05-Goethe-B1-Sprechen-Teil2-recording-note.md
```

## 复盘要记录什么

- 模块分数和时间压力。
- 每个错题的原文证据。
- 听力错因：数字、人物、转折、同义替换。
- 写作错因：任务点、格式、语序、词尾。
- 口语错因：卡顿词、互动不足、理由不够。

## 不要做

- 不把官方 PDF、音频下载后提交到公开仓库。
- 不把同一套题刷到背答案。
- 不用 telc/OESD 的题型替代 Goethe 题型。
- 不只记录分数，不记录错因。
