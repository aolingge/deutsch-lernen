# B1 听力解题策略

B1 听力最怕“听到一个词就立刻选”。更稳的做法是先判断场景，再抓时间、地点、数字、否定和转折。

## Part 1：电话对话

常见场景：

- 预约医生或牙医
- 改约或取消预约
- 咨询课程和报名活动
- 询问价格或费用
- 确认地址、时间、需要携带的材料

听前预测问题：

- `Wann findet der Termin statt?` 预约什么时候？
- `Was muss man mitbringen?` 需要带什么？
- `Wie viel kostet es?` 多少钱？
- `Warum kann die Person nicht kommen?` 为什么不能来？

重点信号：

- 日期和时间：`Datum`, `Uhrzeit`
- 地点：`Ort`, `Adresse`
- 数字：价格、电话号码、房间号
- 否定：`nicht`, `kein`, `leider`

## Part 2：广播与公告

常见场景：

- 火车站或公交通知
- 天气预报和天气预警
- 道路施工、事故、堵车
- 活动取消或推迟
- 营业时间变化
- 公共设施通知

高频词：

| Deutsch | 中文 |
| --- | --- |
| verschieben | 改期 |
| absagen | 取消 |
| mitbringen | 携带 |
| bezahlen | 支付 |
| ausfallen | 取消、停运 |
| umfahren | 绕行 |
| die Baustelle | 工地、施工点 |
| der Stau | 堵车 |
| wegen + Genitiv/Dativ | 因为…… |
| nicht ..., sondern ... | 不是……而是…… |

## Part 3：短对话

常见题型：

- 观点态度题：喜欢、不喜欢、支持、反对
- 细节信息题：地点、原因、价格、时间
- 否定转折题：前面说想做某事，后面因为原因改变决定

听到这些词要特别注意：

- `aber`
- `trotzdem`
- `leider`
- `weil`
- `Ich finde, dass ...`
- `Meiner Meinung nach ...`
- `Ich habe keine Lust auf ...`

## Part 4：长独白或访谈

常见内容：

- 人物采访
- 经验分享
- 科普小知识
- 活动介绍
- 职场报告

解题重点：

- 顺序词：`erstens`, `zweitens`, `zuletzt`
- 递进词：`ausserdem`
- 因果词：`deshalb`, `weil`
- 转折词：`aber`, `trotzdem`
- 数字细节：年份、时长、数量、价格

## 练习方法

1. 第一遍只判断场景和人物关系。
2. 第二遍记录时间、地点、数字和否定词。
3. 做完后整理同义替换，而不是只看答案。
