# B1 阅读解题策略

阅读不是逐词翻译比赛。B1 阅读更看重你能否快速定位信息、识别态度、看懂规则。

## Teil 1：通信判断正误

题型特点：

- 常见材料是邮件、留言、短消息或私人通信。
- 题目多问人物、时间、安排、原因和是否发生。
- 重点是判断题目是否忠实表达原文信息。

解题步骤：

1. 先读题目，圈人物、时间、动作和否定词。
2. 回原文找同义表达，不要只找完全一样的词。
3. 对比程度词：`immer`, `nie`, `alle`, `nur`, `manchmal`。
4. 题目把“可以”改成“必须”时，不要选对。

## Teil 2：信息或论证文本三选一

题型特点：

- 文本比 Teil 1 更长，常是报道、说明、观点或经验。
- 选项可能有原文词，但逻辑关系不一定对。

解题步骤：

1. 先看问题问的是主旨、细节、原因还是观点。
2. 每段读第一句和转折词附近内容。
3. 找到证据句后，再比较三个选项。
4. 不认识的词先跳过，优先看动词、连接词和否定词。

## Teil 3：情境匹配

题型特点：

- 人物需求和广告、活动、课程、服务信息匹配。
- 关键词通常是时间、地点、价格、年龄、对象、是否需要报名。

解题步骤：

1. 给每个人写 2-3 个硬条件。
2. 先排除明显不满足条件的选项。
3. 主题相似但时间、对象、费用不符，也要排除。
4. 最后再处理可能适合多个选项的题。

## Teil 4：观点匹配

题型特点：

- 多个短段落，每段表达一个人的观点。
- 题目通常问这个人是否支持某个行为、禁令或活动。
- 核心是判断态度：支持、反对、保留意见。

解题步骤：

1. 先看题目，圈出判断对象。
2. 读段落，找态度词和转折词。
3. 特别看 `aber`, `dennoch`, `trotzdem`, `allerdings` 后面的内容。
4. 和题目逐一比对，不要只凭一个关键词选答案。

## 态度词

支持：

- `Ich stimme zu.`
- `Ich halte es fuer sinnvoll.`
- `Ich finde es gut.`
- `Das kommt in Frage.`

反对：

- `Ich bin dagegen.`
- `Ich widerspreche.`
- `Ich halte es fuer unlogisch.`
- `Das ist fuer mich keine gute Idee.`

保留意见：

- `Das haengt davon ab.`
- `Man muss zuerst pruefen, ...`
- `Ich bin mir nicht sicher.`

## Teil 5：规则说明类选择题

题型特点：

- 内容常见于博物馆、场馆、课程、会员规则、活动说明。
- 题目多问许可、禁止、费用、优惠、时间、报名条件。
- 需要关键词定位和细节比对。

解题步骤：

1. 先看题目，圈出关键词，例如 `Fuehrung`, `Mitglied`, `Ermaessigung`。
2. 回到原文找同词或同义词。
3. 比对选项里的时间、条件、费用和否定词。
4. 注意 `nur`, `erst`, `bis`, `ab`, `kostenlos`, `nicht erlaubt`。
