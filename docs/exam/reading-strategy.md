# B1 阅读解题策略

阅读不是逐词翻译比赛。B1 阅读更看重你能否快速定位信息、识别态度、看懂规则。

## Teil 4：观点匹配

题型特点：

- 多个短段落，每段表达一个人的观点。
- 题目通常问这个人是否支持某个行为、禁令或活动。
- 核心是判断态度：支持、反对、保留意见。

解题步骤：

1. 先看题目，圈出判断对象。
2. 读段落，找态度词和转折词。
3. 特别看 `aber`, `dennoch`, `trotzdem`, `allerdings` 后面的内容。
4. 和题目逐一比对，不要只凭一个关键词选答案。

## 态度词

支持：

- `Ich stimme zu.`
- `Ich halte es fuer sinnvoll.`
- `Ich finde es gut.`
- `Das kommt in Frage.`

反对：

- `Ich bin dagegen.`
- `Ich widerspreche.`
- `Ich halte es fuer unlogisch.`
- `Das ist fuer mich keine gute Idee.`

保留意见：

- `Das haengt davon ab.`
- `Man muss zuerst pruefen, ...`
- `Ich bin mir nicht sicher.`

## Teil 5：规则说明类选择题

题型特点：

- 内容常见于博物馆、场馆、课程、会员规则、活动说明。
- 题目多问许可、禁止、费用、优惠、时间、报名条件。
- 需要关键词定位和细节比对。

解题步骤：

1. 先看题目，圈出关键词，例如 `Fuehrung`, `Mitglied`, `Ermaessigung`。
2. 回到原文找同词或同义词。
3. 比对选项里的时间、条件、费用和否定词。
4. 注意 `nur`, `erst`, `bis`, `ab`, `kostenlos`, `nicht erlaubt`。
