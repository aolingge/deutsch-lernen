# B2-C1 学术表达库

这页服务 B2、C1、DSH、TestDaF 和 telc C1 Hochschule。不要把表达当万能模板硬套；先确定段落功能，再选句型。

## 摘要 Zusammenfassung

```text
Der Text behandelt die Frage, ob ...
Im Mittelpunkt steht ...
Der Autor weist darauf hin, dass ...
Zunaechst wird beschrieben, dass ...
Anschliessend geht der Text auf ... ein.
Abschliessend wird deutlich, dass ...
```

用法：摘要只保留核心信息，不加入个人评价。

## 图表描述 Grafikbeschreibung

```text
Die Grafik zeigt ...
Die Angaben beziehen sich auf ...
Auffaellig ist, dass ...
Im Vergleich dazu ...
Der Anteil steigt / sinkt / bleibt stabil.
Aus der Darstellung laesst sich ableiten, dass ...
```

常见错误：

- 不要把每个数字都念一遍。
- 先说趋势，再说例外。
- 不确定原因时用 `moeglicherweise` 或 `dies koennte darauf hindeuten, dass ...`。

## 论证 Argumentation

```text
Ein zentrales Argument dafuer ist ...
Dafuer spricht ausserdem, dass ...
Dies laesst sich damit begruenden, dass ...
Ein Beispiel hierfuer ist ...
Daraus folgt jedoch nicht automatisch, dass ...
```

段落结构：

```text
These -> Grund -> Beispiel -> Einschraenkung -> Rueckbezug
```

## 让步 Konzession

```text
Zwar ist richtig, dass ..., jedoch ...
Obwohl ... eine wichtige Rolle spielt, ...
Auch wenn ... nachvollziehbar ist, bleibt problematisch, dass ...
Man muss einraeumen, dass ...
Gleichzeitig sollte beruecksichtigt werden, dass ...
```

用法：让步不是自我否定，而是让你的立场更可信。

## 反驳 Einwand und Gegenposition

```text
Gegen diese Sichtweise laesst sich einwenden, dass ...
Diese Argumentation ueberzeugt nur teilweise, weil ...
Problematisch an dieser Position ist ...
Dem kann entgegengehalten werden, dass ...
Der Einwand ist berechtigt, fuehrt aber nicht zwingend zu ...
```

## 原因、结果和限制

```text
Dies ist vor allem darauf zurueckzufuehren, dass ...
Eine moegliche Ursache liegt in ...
Infolgedessen ...
Dies hat zur Folge, dass ...
Unter bestimmten Voraussetzungen ...
Dies gilt allerdings nur, wenn ...
```

## 引用和转述

```text
Der Autor betont, dass ...
Im Text wird die Auffassung vertreten, dass ...
Es wird darauf hingewiesen, dass ...
Nach Ansicht der Autorin ...
Die Studie kommt zu dem Ergebnis, dass ...
```

Konjunktiv-I 识别：

```text
Der Autor sagt, die Massnahme sei notwendig.
Die Studie zeige, dass ...
```

如果你还不稳定，先用 `laut dem Text` 和 `der Text zeigt`，比乱用 Konjunktiv 更安全。

## 结论 Schluss

```text
Zusammenfassend laesst sich sagen, dass ...
Insgesamt ueberwiegen die Vorteile / Nachteile.
Entscheidend ist daher nicht ..., sondern ...
Aus diesem Grund erscheint es sinnvoll, ...
Weitere Untersuchungen waeren noetig, um ...
```

## DSH/TestDaF 使用提醒

- 写作：每段先写功能词，再扩展内容。
- 口语：表达要短而稳，追问时不要背长句。
- 阅读：把这些表达当信号词识别文本结构。
- 听力：听到 `entscheidend ist`, `problematisch ist`, `daraus folgt` 时立刻记结构。
