# TestDaF 与 C1 衔接

如果你还没确定要考 TestDaF、DSH 还是 telc C1 Hochschule，先看 [德语考试选择树](exam-decision-tree.md)。

`TestDaF = Test Deutsch als Fremdsprache / 德福考试 / 面向希望在德国高校学习的德语学习者。`

TestDaF 不是单纯的 C1 语法考试，而是把阅读、听力、写作、口语分开评为 TDN 3、TDN 4、TDN 5。很多德语授课大学常见目标是四项 TDN 4，但具体要求必须查目标大学。

## TDN 3/4/5 怎么用

| 等级 | 备考理解 |
| --- | --- |
| TDN 3 | 接近入门大学语言能力，但很多专业不够 |
| TDN 4 | 常见大学入学目标 |
| TDN 5 | 高于常规要求，说明该项能力很强 |

注意：有些大学会接受不同组合或允许补偿，有些专业要求更高。永远以目标大学页面为准。

## C1 能力如何转成 TestDaF 动作

| C1 能力 | TestDaF 动作 |
| --- | --- |
| 读懂长文本 | 快速判断段落功能、作者态度、细节证据 |
| 听懂讲座/访谈 | 做结构笔记，区分主论点和例子 |
| 写清楚论证 | 根据材料和任务写出结构化文本 |
| 口头表达观点 | 在时间压力下组织回答并给例子 |

## 备考路线

### B2 后期

- 复习复杂句、被动、名词化、连接词。
- 每周做 2 次新闻摘要。
- 口语从 2 分钟扩展到 4 分钟。

### C1 初期

- 读长文本，做段落功能标注。
- 听讲座，做 Cornell note 或表格笔记。
- 写 250 词左右论证文。

### TestDaF 专项

- 阅读：限时定位，不逐词翻译。
- 听力：先预测题型，再做结构笔记。
- 写作：固定引入、主体、结论和材料引用方式。
- 口语：每题先用固定开头争取组织时间。

## 和 DSH 的选择

| 情况 | 更可能适合 |
| --- | --- |
| 你想要标准化考试、全球考点 | TestDaF |
| 你已经在德国大学语言中心学习 | DSH |
| 你写作和口语抗压弱 | 先比较样题，选择反馈更容易获得的考试 |
| 目标大学明确列出一种考试 | 跟目标大学要求走 |
