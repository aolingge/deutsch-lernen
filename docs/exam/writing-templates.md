# B1 写作模板

写作模板的目标不是机械套话，而是先保证结构稳定、语法少错、要点不漏。

## 总体检查清单

写完后按这个顺序检查：

1. 题干要点是否全部覆盖。
2. 称呼和结尾是否符合对象关系。
3. 陈述句动词是否在第二位。
4. 从句动词是否在句尾。
5. 名词首字母是否大写。
6. 冠词、格和形容词词尾是否明显错误。

## Teil 1：私人邮件

适用对象：朋友、熟人、同辈，用 `du`。

结构：

```text
Lieber/Liebe [Name],

wie geht es dir? Ich schreibe dir, weil ...

[要点 1]
[要点 2]
[要点 3]

Schreib mir bitte bald zurueck.

Liebe Gruesse
[Name]
```

常用句：

```text
Vielen Dank fuer deine Nachricht.
Ich habe mich sehr darueber gefreut.
Leider kann ich nicht kommen, weil ...
Wollen wir uns am Samstag treffen?
Ich freue mich auf deine Antwort.
```

## Teil 2：论坛观点帖

结构：

```text
Das Thema meines Kommentars ist: ...

Meiner Meinung nach ...
Der Grund ist, dass ...
Zum Beispiel ...

Einerseits ..., andererseits ...
Trotzdem finde ich, dass ...

Zum Schluss moechte ich sagen, dass ...
```

常用连接词：

- `zuerst`：首先
- `ausserdem`：此外
- `deshalb`：因此
- `trotzdem`：尽管如此
- `zum Beispiel`：例如
- `meiner Meinung nach`：在我看来

## Teil 3：正式或半正式邮件

适用对象：机构、老师、陌生人，用 `Sie`。

结构：

```text
Sehr geehrte Damen und Herren,

ich schreibe Ihnen, weil ...

[说明情况]
[提出请求 / 道歉 / 询问]

Ich freue mich auf Ihre Antwort.

Mit freundlichen Gruessen
[Name]
```

常用句：

```text
Ich moechte Sie bitten, ...
Koennten Sie mir bitte sagen, ...?
Leider muss ich den Termin absagen.
Waere es moeglich, einen neuen Termin zu vereinbaren?
Vielen Dank fuer Ihre Hilfe.
```

## 一套万能小骨架

观点类写作可以先练这个四步：

1. 观点：`Ich denke, dass ...`
2. 理由：`Der Grund ist, dass ...`
3. 例子：`Zum Beispiel ...`
4. 结论：`Deshalb finde ich, dass ...`

例句：

```text
Ich denke, dass oeffentliche Verkehrsmittel sehr wichtig sind.
Der Grund ist, dass sie umweltfreundlicher sind.
Zum Beispiel kann man mit dem Bus oder Zug fahren und Benzin sparen.
Deshalb finde ich, dass man oefter oeffentliche Verkehrsmittel benutzen sollte.
```
