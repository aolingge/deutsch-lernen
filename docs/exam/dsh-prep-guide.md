# DSH 备考指南

`DSH = Deutsche Sprachprüfung für den Hochschulzugang / 德国高校入学德语考试 / 用来证明非母语申请者具备德语授课大学学习能力。`

DSH 不是普通生活德语考试。它更像“大学学习能力测试”：你要听懂讲座、读懂学术或准学术文本、处理语法结构、写出论证，并能口头表达学术话题。

Last checked: 2026-05-05

如果你还没确定要考 DSH、TestDaF 还是 telc C1 Hochschule，先看 [德语考试选择树](exam-decision-tree.md)。

## DSH-1 / DSH-2 / DSH-3 怎么理解

| 等级 | 常见门槛 | 大致意义 | 备考判断 |
| --- | ---: | --- | --- |
| DSH-1 | 笔试和口试均至少 57% | 低于多数德语授课专业常见要求 | 可能只适用于部分预备或特殊条件，必须查大学 |
| DSH-2 | 笔试和口试均至少 67% | 很多德语授课专业常见目标 | 备考主目标通常设为 DSH-2 |
| DSH-3 | 笔试和口试均至少 82% | 很高水平 | 适合语言要求高或想证明更强学术德语的人 |

重要：不同大学和专业可能有自己的要求。不要用“别人学校接受”推断你的目标学校。

## 常见模块

DSH 形式由各注册高校执行，题型会有差异，但通常围绕这些能力：

| 模块 | 考什么 | 训练动作 |
| --- | --- | --- |
| Hoerverstehen | 听讲座或较长说明并答题 | 做结构笔记：主题、论点、例子、数据 |
| Leseverstehen | 阅读较长学术/说明文本 | 标段落功能和论证关系 |
| Wissenschaftssprachliche Strukturen | 学术语言结构 | 改写、名词化、被动、连接、从句 |
| Textproduktion | 写作 | 摘要、图表、评论、论证 |
| Muendliche Pruefung | 口试 | 简短报告、解释图表、回答追问 |

公开框架中常见的笔试权重为 HV:LV:WS:TP = 2:2:1:2。不同考点样题风格不同，备考时要看目标考点或目标大学语言中心说明。

## 常见时间框架

| 模块 | 常见时间 |
| --- | --- |
| Hoerverstehen | 听两遍；第一遍后约 10 分钟、第二遍后约 40 分钟处理 |
| Leseverstehen + WS | 约 90 分钟 |
| Textproduktion | 约 70 分钟 |
| Muendliche Pruefung | 口试最多约 20 分钟，另有准备时间 |

## 从 B2 到 DSH 的路线

### 阶段一：B2 补足

- 主题：教育、科学、社会、媒体、环境、技术、经济。
- 语法：被动、名词化、关系从句、连接副词、Konjunktiv II。
- 输出：每周一篇 180-250 词观点文。

### 阶段二：C1 学术化

- 每周读 2 篇长文本。
- 每周听 1 段讲座或访谈并做笔记。
- 每周写 1 篇摘要或论证文。
- 每周做 1 次 5-8 分钟口头报告。

### 阶段三：DSH 模块化

- 听力：练缩写、结构笔记、数字和例子。
- 阅读：练同义改写、段落功能、定义和限制条件。
- 结构：练句子改写，不只背规则。
- 写作：固定“引入-描述-分析-评价-结论”骨架。
- 口试：固定“主题-定义-数据/例子-利弊-个人评价”骨架。

## 和 TestDaF / telc C1 Hochschule 的关系

- DSH 通常在大学或 Studienkolleg 体系内进行，风格可能更接近当地大学语言中心课程。
- TestDaF 是标准化考试，四项成绩以 TDN 3/4/5 呈现。
- telc Deutsch C1 Hochschule 偏大学入学和学术职业场景。
- 选择哪个考试，先看目标大学承认清单，再看你的强弱项。
