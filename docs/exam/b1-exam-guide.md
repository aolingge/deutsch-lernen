# B1 考试备考

## 考试能力目标

B1 重点考察你能否在常见场景中：

- 理解文章或对话的大意
- 找到具体信息
- 表达观点和理由
- 写清楚邮件、留言或短文
- 用简单但完整的句子交流

## 阅读

做题顺序建议：

1. 先读题干，圈关键词。
2. 再读文章，找同义替换。
3. 不要因为一个陌生词停太久。
4. 最后检查否定词和时间词。

## 听力

听前预测：

- 谁在说话？
- 场景在哪里？
- 可能会出现哪些数字、时间、地点？
- 问题要找事实还是态度？

## 写作

基础结构：

```text
Anrede
Einleitung
Hauptteil mit 2-3 Punkten
Schluss
Gruss
Name
```

常用句：

```text
Vielen Dank fuer Ihre Nachricht.
Ich moechte Sie bitten, ...
Leider kann ich ...
Ich freue mich auf Ihre Antwort.
```

## 口语

口语练习先追求稳定表达：

- Ich bin der Meinung, dass ...
- Ein Vorteil ist, dass ...
- Ein Nachteil ist, dass ...
- Ich stimme dir zu.
- Ich sehe das anders, weil ...
