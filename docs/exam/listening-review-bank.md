# B1 听力例题复盘库

听力错题要按错因分类，不要只写“没听懂”。本页用于收集 Teil 1-4 的典型错法和可复用听力动作。

## 复盘字段

```text
日期：
来源：
Teil：
题目问什么：
我听到的关键词：
正确关键词：
错因：
下次动作：
```

## Teil 1：短公告/留言

常见错因：

- 数字和时间听错。
- 没听到地点变化。
- 只听到一个关键词就选。

下次动作：

```text
听前圈数字/时间/地点。
听到 leider/aber 之后重新判断。
```

## Teil 2：较长信息

常见错因：

- 第一遍没抓主题。
- 第二遍陷入细节。
- 同义替换没认出。

下次动作：

```text
第一遍只写主题和场景。
第二遍再找选项证据。
```

## Teil 3：对话判断

常见错因：

- 混淆说话人。
- 把建议当成已经发生。
- 没听出否定。

下次动作：

```text
给每句话标 A/B 说话人。
听 sollen/koennen/muessen 判断任务还是事实。
```

## Teil 4：多人观点/匹配

常见错因：

- 态度词没听出。
- 转折后态度改变。
- 人物顺序混乱。

下次动作：

```text
每个人只贴一个态度标签：支持/反对/担心/建议。
听到 aber/trotzdem/doch 重判态度。
```
