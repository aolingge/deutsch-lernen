# 学习资源组合包

资源不是越多越好。每个阶段只保留一个主线、一个练习站、一个输入源、一个输出任务。

| --- | --- |

## 资源减法规则

如果你已经有主线课程，不要再同时开 3 个同类课程。保留资源的标准：

- 我本周会用它。
- 它解决一个明确问题。
- 它来源清楚、可公开访问。
- 它不会诱导我下载来源不明材料。
