# 德语语法外链资源库

本页负责跳到外站学习细节。本仓库保存学习路线和练习动作，外站负责完整解释、例句和题库。

Last checked: 2026-05-05

## 总入口

| 资源 | 链接 | 适合程度 | 怎么用 |
| --- | --- | --- | --- |
| DW Learn German | [learngerman.dw.com](https://learngerman.dw.com/en/overview) | 官方媒体学习平台，A1-C1 | 用 Nicos Weg 做 A1-B1 主线，每课后补语法 |
| Goethe-Institut 免费练习 | [goethe.de/spr/ueb](https://www.goethe.de/en/spr/ueb.html) | 官方机构，适合 A1-B1 练习 | 做 Goethe 风格练习，考试前优先 |
| FU Berlin DaF-Portale | [FU Berlin resource list](https://www.sprachenzentrum.fu-berlin.de/slz/sprachen-links/deutsch/daf_portale/index.html) | 大学语言中心推荐清单 | 用来找和验证可靠学习入口 |
| Grimm Grammar | [COERLL / UT Austin](https://coerll.utexas.edu/gg/pr/index.html?si=pro) | 大学公开语法站，英文解释 | 查 cases、verbs、determiners、subjunctive、passive |
| Dartmouth German Grammar Review | [Dartmouth grammar](https://germanstudiesdepartmenaluser.host.dartmouth.edu/) | 大学语法复习站 | 深挖词序、从句、格、虚拟式 |
| University of Michigan German Grammar | [Michigan grammar](https://resources.german.lsa.umich.edu/grammatik/) | 大学课程资源 | 介词、格、词序、时态专项训练 |
| Lingolia German Grammar | [Lingolia grammar](https://deutsch.lingolia.com/en/grammar) | 成熟语法站 | 每个语法点读规则，然后做免费练习 |
| mein-deutschbuch Grammatik | [mein-deutschbuch.de](https://mein-deutschbuch.de/grammatik.html) | 德语解释，目录很全 | B1 后期用德语查语法 |
| Schubert-Verlag Online-Aufgaben | [Schubert exercises](https://www.schubert-verlag.de/aufgaben/) | 教材出版社练习 | 学完规则后刷分级练习 |
| IDS grammis | [grammis.ids-mannheim.de](https://grammis.ids-mannheim.de/) | 权威参考语法 | 查证争议和细节，不做每日主线 |

| 形容词词尾/比较级 | [Lingolia adjectives](https://deutsch.lingolia.com/en/grammar/adjectives), [Grimm](https://coerll.utexas.edu/gg/pr/index.html?si=pro), [Dartmouth](https://germanstudiesdepartmenaluser.host.dartmouth.edu/) | 按定冠词、不定冠词、无冠词三表练 |
| 代词 | [Lingolia pronouns](https://deutsch.lingolia.com/en/grammar/pronouns), [Grimm](https://coerll.utexas.edu/gg/pr/index.html?si=pro) | 代词和格绑定，关系代词单独练 |

| 介词动词 | [Michigan prepositional verbs](https://resources.german.lsa.umich.edu/grammatik/prepositional-verbs/), [mein-deutschbuch](https://mein-deutschbuch.de/grammatik.html), [IDS grammis](https://grammis.ids-mannheim.de/) | 做 Anki：`warten auf + A`, `teilnehmen an + D` |
| 词序/句框 | [Dartmouth main clauses](https://germanstudiesdepartmenaluser.host.dartmouth.edu/WordOrder/MainClauses.html), [Lingolia sentence structure](https://deutsch.lingolia.com/en/grammar/sentence-structure), [Michigan Wortstellung](https://sites.lsa.umich.edu/german-resources/grammatik/wortstellung/) | 标 V2、句末不定式/分词、TeKaMoLo |
| 从句 | [Dartmouth dependent clauses](https://germanstudiesdepartmenaluser.host.dartmouth.edu/WordOrder/Dependent.html), [Lingolia sentence structure](https://deutsch.lingolia.com/en/grammar/sentence-structure), [mein-deutschbuch](https://mein-deutschbuch.de/grammatik.html) | 每类从句记“连词-动词位置-功能-例句” |
| Infinitiv mit zu / um...zu | [Grimm](https://coerll.utexas.edu/gg/pr/index.html?si=pro), [Lingolia sentence structure](https://deutsch.lingolia.com/en/grammar/sentence-structure), [mein-deutschbuch](https://mein-deutschbuch.de/grammatik.html) | 和目的句、替代句一起整理 |
| 被动 Passiv | [Lingolia verbs](https://deutsch.lingolia.com/en/grammar/verbs), [Grimm](https://coerll.utexas.edu/gg/pr/index.html?si=pro), [mein-deutschbuch](https://mein-deutschbuch.de/grammatik.html) | 先掌握 `werden + Partizip II`，再识别 Zustandspassiv |
| Konjunktiv II | [Grimm subjunctive](https://coerll.utexas.edu/gg/pr/index.html?si=pro), [Dartmouth](https://germanstudiesdepartmenaluser.host.dartmouth.edu/), [mein-deutschbuch](https://mein-deutschbuch.de/grammatik.html) | 先学 `wuerde + Infinitiv`，再学 waere/haette/koennte/sollte |
| 否定 | [Grimm negation](https://coerll.utexas.edu/gg/pr/index.html?si=pro), [Dartmouth](https://germanstudiesdepartmenaluser.host.dartmouth.edu/), [mein-deutschbuch](https://mein-deutschbuch.de/grammatik.html) | 重点练 nicht 位置和 kein 词尾 |
| 分级练习 | [Schubert](https://www.schubert-verlag.de/aufgaben/), [DeutschAkademie](https://www.deutschakademie.de/online-deutschkurs/), [Lingolia](https://deutsch.lingolia.com/en/grammar) | 每学一个点做 10-20 题，错题入库 |

## 使用顺序

```text
先看本仓库路线 -> 去 Lingolia/Grimm/Dartmouth 查规则 -> 去 Schubert/DeutschAkademie 做题 -> 把错题写回自己的错题库。
```
