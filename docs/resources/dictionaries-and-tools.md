## 查词流程

遇到一个新词时，推荐这样查：

1. 先用 PONS、LEO 或 dict.cc 快速看中文/英文大意。
2. 再用 Duden 或 DWDS 查词性、冠词、复数、常见搭配。
3. 把完整短语记下来，而不是只记单词。
4. 自己造一个 B1 难度例句。

示例：

```text
der Termin, -e
einen Termin vereinbaren
Ich moechte gern einen Termin vereinbaren.
```

## 工具风险提醒

- 机器翻译可以帮理解，但不能直接复制进写作作业。
- 公开 Anki 卡组质量参差不齐，使用前先抽查 20 张。
- 不要上传自己从教材、课程或付费 App 里导出的词库。
