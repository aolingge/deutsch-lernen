# B1 学习网站与练习资源

最稳的资源组合是：Goethe 官方材料校准考试，DW/vhs 做主线，Schubert/Lingolia 做语法练习，Easy German/Slow German 做真实输入，Anki 做复习。

Last checked: 2026-05-05

## 资源组合建议

| 阶段 | 主资源 | 每周动作 |
| --- | --- | --- |
| B1 初期 | DW Nicos Weg + vhs + Lingolia | 主线课 4 天，语法练习 3 天 |
| B1 中期 | Schubert + Easy German + Goethe 词表 | 语法刷题，听读输入，主题词输出 |
| 考前 4 周 | Goethe 官方练习 + 模考模板 | 每周至少 1 次限时模块或整套模考 |

## 不建议

- 不用“包过/押题/内部题库”当主线。
- 不把 telc/ÖSD 当成 Goethe 的替代真题。
- 不下载和转发盗版教材、音频、课程录屏。
