# 如何筛选资源

学习资源越多，越容易收藏过量、执行不足。这页帮助你判断一个资源是否值得放进学习计划。

## 推荐优先级

1. 官方资源：Goethe、DW、BAMF、telc、vhs、TestDaF、OeSD。
2. 长期维护的教育项目：Easy German、Slow German、Lingolia、Schubert Verlag。
3. 教师原创频道：有清晰课程结构、例句、等级标注和稳定更新。
4. 平台搜索结果：YouTube、B 站、小红书、论坛，只作为线索。
5. 来源不明资源包：谨慎使用，不进入公开仓库。

## 一个好资源应该具备什么

- 标明适合级别，例如 A1、A2、B1。
- 有清楚的练习目标，例如听力、语法、词汇、写作。
- 有例句或练习，不只是讲概念。
- 内容来源清楚，版权风险低。
- 能被持续使用，而不是只适合刷一晚上。

## 什么时候不要加入仓库

- 需要加私人联系方式才能领取。
- 来源写着“内部资料”“真题泄露”“机构押题”。
- 大量复制教材、课程讲义、字幕或付费材料。
- 页面里夹带个人信息、聊天记录、私人网盘、二维码。
- 内容过时，或者只靠夸张标题吸引点击。

## 收藏到学习计划的格式

建议每个资源只记录 4 件事：

```text
资源名：
链接：
适合阶段：
我准备用它解决什么问题：
```

例子：

```text
资源名：Slow German
链接：https://slowgerman.com/
适合阶段：B1+
我准备用它解决什么问题：每周练 2 次慢速听力和复述。
```

## 每周资源使用配比

如果只有 7-10 小时学习时间，可以这样分：

| 类型 | 时间 | 示例 |
| --- | --- | --- |
| 主线课程 | 40% | DW Nicos Weg / vhs-Lernportal |
| 语法和词汇练习 | 25% | Lingolia / Schubert / Anki |
| 听力和视频 | 25% | Easy German / Slow German / Goethe 24h Deutsch |
| 输出练习 | 10% | B1 写作和口语模板 |

## 维护者检查清单

新增资源前确认：

- 链接可以公开访问。
- 资源不要求泄露私人信息。
- 没有复制受版权保护的正文、字幕、PDF 或视频。
- 仓库只写简介和学习建议，不镜像外部内容。
- 已运行 `pwsh ./tools/privacy-scan.ps1 .`。
