# 官方课程与系统学习平台

这些资源适合当作学习主线。主线的意思是：它能告诉你下一课学什么，有练习，有难度梯度，不只是零散视频。

Last checked: 2026-04-21

## DW Learn German

- Link: [DW Learn German](https://learngerman.dw.com/de/deutsch-lernen/s-9095)
- 适合：A1-B1 自学主线，尤其是喜欢视频剧情和交互练习的人。
- 重点资源：`Nicos Weg`、等级测试、课程练习、听力材料。
- 使用方法：按 A1、A2、B1 顺序走，每课至少完成视频、词汇、语法和练习，不建议只看 YouTube 电影合集。

## Goethe-Institut 免费练习

- Link: [Kostenlos Deutsch ueben](https://www.goethe.de/de/spr/ueb.html)
- 适合：补练语法、词汇、阅读、听力、生活主题和文化内容。
- 亮点：Goethe 的资源按主题和等级整理，覆盖 A1 到 C 级别。
- 使用方法：每周选 1 个主题，例如 `Leben & Alltag` 或 `Arbeiten & Beruf`，把里面的词汇和句型整理到自己的复习表。

## Goethe: Leben & Alltag

- Link: [Leben & Alltag](https://www.goethe.de/de/spr/ueb/lua.html)
- 适合：准备德国生活场景的人，例如住房、购物、工作、办事。
- 亮点：把语言练习和德国生活场景放在一起，适合从“考试德语”过渡到“生活德语”。
- 使用方法：配合本仓库的 [德国生活指南](../life-in-germany/README.md) 使用。

## vhs-Lernportal

- Link: [vhs-Lernportal Kursangebot](https://www.vhs-lernportal.de/wws/kursangebot-lernende.php)
- 适合：想要大量练习、按课程节奏推进的人。
- 推荐入口：
  - [A2-Deutschkurs](https://a2.vhs-lernportal.de/wws/home.php)
  - [B1-Deutschkurs](https://b1.vhs-lernportal.de/wws/home.php)
  - [A2-B1 Beruf](https://a2-b1-beruf.vhs-lernportal.de/wws/home.php)
- 使用方法：把 vhs 当作练习量来源，每天固定做 20-30 分钟，不要一次刷太多。

## Lingolia German

- Link: [Lingolia German](https://deutsch.lingolia.com/en/)
- 适合：查语法解释、做少量交互练习、补主题词汇。
- 亮点：语法解释清楚，通常有例句、翻译和基础练习。
- 使用方法：遇到一个语法点卡住时，先看 Lingolia 解释，再回到主线课程做题。

## Schubert Verlag Online-Aufgaben

- Link: [Schubert Verlag Online-Aufgaben](https://schubert-verlag.de/aufgaben/arbeitsblaetter_a1_z/a1_arbeitsblaetter_index_z.htm)
- 适合：补 A1-A2 基础练习，也适合复习词序、格、动词变位。
- 使用方法：把它当作“语法刷题本”，按主题选择，不必按页面顺序全做。

## Deutsch Akademie

- Link: [DeutschAkademie Online-Deutschkurs](https://www.deutschakademie.de/online-deutschkurs/)
- 适合：想多做语法题和等级练习的人。
- 使用方法：适合作为第二练习来源，不建议同时开太多练习站，避免只收藏不完成。
