# 从私人 Obsidian 到公开仓库

本仓库可以接收从私人 Obsidian 学习库提炼出来的公开内容，但不接收原始笔记、课堂记录、截图、聊天记录或个人报名信息。

## 推荐流程

1. 在私人 Obsidian 里先整理学习路线和错题。
2. 选出可公开复用的主题，例如 B1 语法清单、资源链接、学习方法。
3. 删除或改写所有个人经历、姓名、账号、地址、付款、报名和课堂群信息。
4. 把 Obsidian wiki 链接改成普通 Markdown 链接。
5. 只链接外部公开来源，不复制受版权保护的大段内容。
6. 运行隐私扫描：

```powershell
./tools/privacy-scan.ps1 .
```

7. 检查 Markdown 链接和目录是否能从 README 到达。

## 适合公开的内容

- 通用语法路线。
- 自己改写的例句和练习方法。
- 官方或公开学习网站链接。
- 错题复盘模板。
- 学习计划和模考复盘流程。

## 不适合公开的内容

- 原始课堂笔记。
- OneNote/Obsidian 原始导入目录。
- 个人考试报名、成绩、身份证件、付款记录。
- 老师或同学的姓名、联系方式、群聊信息。
- 付费课程、教材、音频、PDF 的全文或大段搬运。

## 内容映射建议

| 私人库内容 | 公开仓库位置 |
| --- | --- |
| B1 学习总入口 | `docs/learning-paths/` |
