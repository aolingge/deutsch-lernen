## 使用原则

- A1-A2：一个主线课程 + 一个练习站。
- B1：官方样题 + 四技能训练 + 错题复盘。
- B2：新闻/访谈 + 观点文 + 口语讨论。
- C1/DSH：长文摘要 + 讲座笔记 + 学术写作。
