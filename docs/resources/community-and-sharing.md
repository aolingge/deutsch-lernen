# 资料分享与社群入口

社群是辅助，不是主线。最有效的分享不是“转发资料包”，而是分享错题、录音、作文和模考复盘，让别人能给反馈。

## 可找的社群类型

| 类型 | 适合做什么 | 注意 |
| --- | --- | --- |
| 语伴 App | 练口语、互改句子 | 先文字/语音，谨慎线下见面 |
| Meetup/线上活动 | 固定口语交流 | 看活动质量和主持人 |
| Reddit/论坛 | 看经验帖、问题型 | 答案质量参差，回官方材料验证 |
| 微信公众号/视频号 | 找中文讲解和考试经验 | 警惕包过、押题、盗版资料 |
| 学习打卡群 | 保持节奏 | 不要被资料噪音带跑 |

## 筛选标准

优先看：

- 近 6-12 个月是否更新。
- 是否引用 Goethe/telc/OESD 官方材料。
- 是否讲清评分标准。
- 是否有真实改文/改口语示例。
- 公众号主体、教师背景、收费边界是否清楚。

直接警惕：

- “内部真题”“押题必中”“包过”。
- 要求发身份证、准考证、付款截图到群。
- 代报名却不让你回 Goethe 或官方合作考点核验。
- 让你转发盗版教材 PDF、音频、课程录屏。
- 只卖资料，不讲训练方法和反馈。

## 最有效的分享内容

每周分享这 4 类，比转发 100 个链接更有用：

1. 一张错因统计：本周错在语序、介词、听力数字还是审题。
2. 一段 60 秒口语录音：主题 + 自评 + 想要别人反馈的问题。
3. 一篇 B1 邮件作文：附题目、限时、自己检查过的版本。
4. 一套模考分数和复盘：模块分数 + 错因 + 下周动作。

## 安全分享边界

可以分享：

- 官方链接。
- 自己的笔记模板。
- 自己整理的错因表。
- 自己写的作文和录音。
- Anki 卡片结构。

不要分享：

- 盗版教材、音频、课程录屏。
- 身份证、护照、准考证、成绩单、付款截图。
- 个人住址、电话、真实考试报名截图。
