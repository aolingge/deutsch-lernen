# 听力与阅读高频词汇

这份词表优先服务 B1 听力和阅读。重点不是一次背完，而是能在题干、公告、对话和规则说明里快速认出。

## 同义替换

B1 阅读和听力经常不直接重复题干原词，而是换一种说法。

| 常见词 | 可能替换 |
| --- | --- |
| wichtig | bedeutend, entscheidend, von grosser Bedeutung |
| oft | regelmaessig, haeufig |
| normalerweise | in der Regel, grundsaetzlich |
| bekommen | erhalten |
| haben | besitzen, verfuegen ueber |
| machen | unternehmen |
| sehr | besonders, ausgesprochen |
| teilnehmen | mitmachen, sich beteiligen |
| brauchen | benoetigen |
