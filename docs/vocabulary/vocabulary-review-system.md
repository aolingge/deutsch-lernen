# 词汇复习闭环

词汇不是“看过就会”。A1-C1 的词汇复习应该从单词升级到搭配、句子、场景和输出。

## 四层复习

| 层级 | 目标 | 例子 |
| --- | --- | --- |
| 单词 | 认识意思、词性、复数或变位 | `der Termin`, `vereinbaren` |
| 搭配 | 知道常一起出现的词 | `einen Termin vereinbaren` |
| 句子 | 能在真实句子里使用 | `Ich moechte einen Termin vereinbaren.` |
| 场景 | 能完成任务 | 预约、改期、取消、询问材料 |

## 每日 15 分钟

```text
3 分钟：复习昨天 10 个词
5 分钟：写 5 个搭配
5 分钟：用其中 3 个词造句
2 分钟：标记最容易忘的词
```

## 每周复盘

| 动作 | 怎么做 |
| --- | --- |
| 听写 | 让音频或同伴读句子，写关键词 |
| 默写 | 看中文场景，写德语表达 |
| 改写 | 把简单句升级成原因句或让步句 |
| 输出 | 用 8-10 个本周词说 2 分钟 |

## A1-A2 词汇

重点：生活高频词和动词变位。

方法：

- 每个名词连冠词一起背。
- 每个动词至少背 `ich` 和 `du` 形式。
- 每个场景准备 3 个固定句。

例子：

```text
der Termin
einen Termin machen
Ich moechte einen Termin machen.
Kann ich den Termin verschieben?
```

## B1 词汇

重点：考试任务、连接词、观点表达。

方法：

- 每个主题准备“问题-原因-建议”三件套。
- 把词汇放进邮件、观点帖、口语陈述。
- 错题里出现的同义替换单独收集。

例子：

```text
das Problem
ein Problem loesen
Meiner Meinung nach sollte man das Problem frueh loesen.
```

## B2-C1 词汇

重点：抽象名词、学术搭配、功能动词结构。

方法：

- 把动词变成名词化表达。
- 每周整理 10 个学术搭配。
- 练摘要和图表时强制复用。

例子：

```text
entscheiden -> die Entscheidung
eine Entscheidung treffen
Die Entscheidung haengt von mehreren Faktoren ab.
```

## 错词卡格式

