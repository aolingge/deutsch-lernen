# B1 固定搭配

固定搭配是 B1 提分很快的一类内容。背单词时不要只背单个词，最好连介词、格和例句一起背。

| 搭配 | 中文 | 例句 |
| --- | --- | --- |
| einen Termin vereinbaren | 约时间 | Wir muessen einen Termin fuer das Gespraech vereinbaren. |
| eine Entscheidung treffen | 做决定 | Ich muss bald eine Entscheidung treffen. |
| an einem Kurs teilnehmen | 参加课程 | Ich nehme an einem Deutschkurs teil. |
| sich fuer etwas interessieren | 对……感兴趣 | Ich interessiere mich fuer deutsche Kultur. |
| sich auf etwas freuen | 期待…… | Ich freue mich auf den Sommerurlaub. |
| sich ueber etwas freuen | 为……感到高兴 | Ich freue mich ueber deine Nachricht. |
| sich um etwas kuemmern | 处理、照顾 | Sie kuemmert sich um die Planung. |
| sich fuer etwas entscheiden | 决定选择…… | Ich habe mich fuer ein Studium entschieden. |
| Angst vor etwas haben | 害怕…… | Viele Lernende haben Angst vor Pruefungen. |
| sich auf eine Pruefung vorbereiten | 准备考试 | Ich bereite mich auf die Deutschpruefung vor. |
| mit dem Zug / Bus / Auto fahren | 坐火车 / 公交 / 汽车 | Ich fahre jeden Tag mit dem Bus. |
| im Internet nach etwas suchen | 在网上搜索…… | Ich suche im Internet nach Informationen. |
| eine E-Mail schreiben | 写邮件 | Ich schreibe eine E-Mail an die Schule. |
| seine Meinung aeussern | 表达观点 | Jeder darf seine Meinung aeussern. |
| ein Problem loesen | 解决问题 | Wir versuchen, das Problem gemeinsam zu loesen. |
| Zeit sparen | 节约时间 | Mit dem Fahrrad kann man Zeit sparen. |
| Geld ausgeben | 花钱 | Ich gebe nicht so viel Geld aus. |
| Erfahrungen sammeln | 积累经验 | Im Praktikum sammelt man wichtige Erfahrungen. |
| Plaene machen | 制定计划 | Wir machen Plaene fuer das Wochenende. |

## 易混表达

- `selbst` / `selber`：都可表示“自己”，`selbst` 更正式。
- `eigen-`：作形容词，表示“自己的”，例如 `mein eigenes Buch`。
- `persoenlich`：个人层面的，例如 `meine persoenliche Meinung`。
- `privat`：私人的、非公开的，例如 `mein Privatleben`。
- `das Land`：国家、国土，也可指乡村。
- `der Staat`：国家机器、政权、制度层面的国家。
- `die Stadt`：城市。
