# B1 自检训练

B1 的核心是任务型表达：能处理邮件、投诉、计划、观点、简单讨论。考试中最重要的不是炫技，而是把任务点完整、清楚、礼貌地完成。

## 听力

任务：

听一段公告、访谈或日常讨论，填表：

| 项目 | 记录 |
| --- | --- |
| 主题 |  |
| 说话人态度 | 赞成 / 反对 / 犹豫 / 提醒 |
| 关键信息 |  |
| 转折词 |  |
| 我不确定的地方 |  |

合格标准：

- 能抓住主旨和态度。
- 能识别同义替换，而不是等原词出现。
- 听不懂细节时能先保住题目任务。

## 阅读

任务：

读一篇 B1 短文或邮件，标出：

- 主题句。
- 每段功能：说明、原因、例子、建议、限制。
- 题目中的同义替换。

合格标准：

- 不逐词翻译也能判断答案区域。
- 能分辨事实、观点、建议。
- 能解释为什么选某个答案。

## 写作

任务：写一封正式邮件，投诉或询问一个问题。

骨架：

```text
Sehr geehrte Damen und Herren,
ich schreibe Ihnen, weil ...
Am ... habe ich ...
Leider ...
Koennten Sie bitte ...
Ich bitte um eine kurze Rueckmeldung.
Mit freundlichen Gruessen
...
```

合格标准：

- 覆盖所有任务点。
- 有开头、背景、问题、请求、结尾。
- 从句动词位置基本正确。

## 口语

任务：准备 3 分钟小陈述，主题任选：学习德语、旅行、健康、工作、媒体。

结构：

```text
1. Thema nennen
2. Eigene Erfahrung
3. Vorteile
4. Nachteile oder Problem
5. Meinung und kurzer Schluss
```

合格标准：

- 能给出至少 2 个理由。
- 遇到追问能用简单句回答。
- 不是背整篇，而是按结构说。

## 语法自检

| 项目 | 目标 |
| --- | --- |
| 四格 | 高频冠词和代词错误能自我发现 |
| 从句 | `weil`, `dass`, `wenn`, `obwohl`, `ob` |
| Infinitiv mit zu | `Ich habe vor, ... zu ...` |
| Konjunktiv II | 礼貌请求和建议 |
| 被动基础 | 能识别 `wird ... gemacht` |
