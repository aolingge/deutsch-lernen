# C1 学术与专业德语路线

C1 的目标是能在大学、专业和复杂公共场景中灵活使用德语。它是 DSH、TestDaF、telc C1 Hochschule 的核心准备阶段。

## 能力目标

- 理解长篇文本和隐含逻辑。
- 听懂讲座、报告和复杂讨论。
- 写摘要、评论、图表描述和论证文。
- 做 5-8 分钟报告并回答追问。

## 学习重点

- 学术阅读：定义、论证、限制条件、反驳。
- 学术听力：结构笔记、例子、数据、结论。
- 学术写作：摘要、材料引用、观点评价。
- 学术口语：报告、澄清、追问、让步、反驳。
- 语法：名词化、被动替代、Konjunktiv I 识别、复杂连接。

## 每周训练

- 一篇长文精读和摘要。
- 一段讲座/访谈听力笔记。
- 一篇 250-350 词写作。
- 一次 5-8 分钟报告录音。

## 进入 DSH/TestDaF 前检查

- 能做讲座结构笔记。
- 能从长文中提取论点和反论点。
- 能写 250 词以上论证文。
- 能在口语中先给结构，再展开细节。
