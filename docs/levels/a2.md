# A2 日常稳定路线

A2 的目标是能处理熟悉场景里的简单信息交换。重点从“短句”升级到“过去经历、日常安排、简单邮件”。

## 能力目标

- 描述家庭、学习、工作、购物、交通、健康。
- 讲简单过去经历。
- 写短邮件、留言、预约和改约。
- 听懂慢速日常对话里的时间、地点、原因。

## 学习重点

- Perfekt：`haben/sein + Partizip II`。
- 情态动词：`koennen`, `muessen`, `duerfen`, `sollen`, `wollen`。
- Dativ/Akkusativ 初步。
- 可分动词。
- 常见时间、地点、方向介词。
- `dass`, `weil`, `wenn` 从句入门。

## 每周训练

- 每天用 Perfekt 讲昨天 5 件事。
- 每周写两封短邮件。
- 每周做两次听力场景记录：人物、地点、时间、原因。
- 每周整理 20 个“动词 + 名词/介词”搭配。

## 升级到 B1 前检查

- 能用 8-10 句讲一次过去经历。
- 能写一封 80 词左右的日常邮件。
- 能解释一个简单原因：`weil ...`。
- 听力中能抓时间、地点、数字、转折。
