# C1 自检训练

C1 的目标是学术和专业表达：能处理抽象材料、长文本、讲座、图表、论证和追问。C1 不等于用很多难词，而是能准确组织复杂信息。

## 听力

任务：

听一段讲座、报告或长访谈，完成讲座笔记：

```text
Oberthema:
Leitfrage:
Definition:
These:
Belege:
Beispiele:
Einschraenkung:
Schlussfolgerung:
```

合格标准：

- 能抓住讲者的结构。
- 能区分定义、论点、例子、限制。
- 能在听后写 120 词摘要。

## 阅读

任务：

读一篇较长说明文或学术介绍，写：

- 3 句摘要。
- 2 个核心概念。
- 2 个论据。
- 1 个你认为需要补充证据的地方。

合格标准：

- 能压缩文本，不复制原句。
- 能看出作者的论证路径。
- 能评价材料的适用范围。

## 写作

任务：写 250-300 词学术化短文，主题可以是教育、技术、健康、环境、城市、就业。

结构：

```text
1. Thema und Fragestellung
2. Material- oder Problembeschreibung
3. Analyse mit Gruenden
4. Gegenargument oder Einschraenkung
5. Schlussfolgerung
```

合格标准：

- 有清楚问题意识，不只是列优缺点。
- 能用名词化、被动替代表达和转述表达。
- 能写出限制条件，例如 `unter bestimmten Voraussetzungen`。

## 口语

任务：做 5-8 分钟报告。

报告卡：

```text
Thema:
Warum ist das Thema relevant?
Zentrale Begriffe:
Daten/Beispiele:
Argumente:
Gegenposition:
Eigene Bewertung:
Offene Frage:
```

追问应对：

```text
Das ist ein wichtiger Punkt.
Ich wuerde das differenzieren.
Ein Beispiel dafuer waere ...
Ganz sicher bin ich mir nicht, aber ...
```

## 语法自检

| 项目 | 目标 |
| --- | --- |
| Konjunktiv I | 能识别和基础使用间接引语 |
| Partizipialattribute | 能识别，输出时先少量使用 |
| 功能动词结构 | `eine Entscheidung treffen`, `zur Verfuegung stehen` |
| 被动替代表达 | `sich lassen`, `sein zu + Infinitiv`, `man` |
| 复杂连接 | `waehrend`, `wohingegen`, `sofern`, `indem` |
