# A1 自检训练

这页用于判断 A1 是否真的能用。A1 不追求复杂表达，核心是：听懂慢速基本信息，说出完整短句，写清楚非常简单的任务。

## 使用方式

每周做一次。不要先查词，先完成，再用词典和语法页复盘。

## 听力

任务：

1. 听一段 A1 课程对话或官方练习。
2. 只记录四类信息：人物、地点、时间、数字。
3. 听第二遍后写出 5 个关键词。

合格标准：

- 能抓到大多数时间、价格、房间号、电话号码片段。
- 能判断对话发生在学校、商店、车站、餐厅或办公室。
- 不需要逐字听懂。

## 阅读

任务：

读一则短通知或简单邮件，回答：

```text
Wer?
Wo?
Wann?
Was soll ich tun?
```

合格标准：

- 能找出人名、地点、时间和行动。
- 能区分 `heute`, `morgen`, `am Montag`, `um ... Uhr`。
- 遇到生词时能先靠格式和关键词推断任务。

## 写作

任务：写 5-8 句自我介绍。

可用骨架：

```text
Ich heisse ...
Ich komme aus ...
Ich wohne in ...
Ich lerne Deutsch.
Ich spreche ...
Mein Hobby ist ...
```

升级写法：

```text
Ich lerne Deutsch, weil ich in Deutschland studieren moechte.
```

合格标准：

- 每句有主语和动词。
- `ich bin`, `ich habe`, `ich komme`, `ich wohne` 不混用。
- 大小写和句号基本稳定。

## 口语

任务：录音 1 分钟。

必须包含：

- 姓名、国家、城市。
- 学习德语的原因。
- 一个爱好。
- 一个简单问题，例如 `Und du?` 或 `Woher kommst du?`

合格标准：

- 能不用中文稿说完。
- 动词大多在第二位。
- 发音不完美没关系，但关键信息清楚。

## 语法自检

| 项目 | 能做到才算过 |
| --- | --- |
| V2 主句 | `Ich lerne Deutsch.` |
| W-Frage | `Wo wohnst du?` |
| Ja/Nein 问句 | `Lernst du Deutsch?` |
| sein/haben | `Ich bin Student.` / `Ich habe Zeit.` |
| nicht/kein | `Ich habe kein Auto.` / `Ich komme nicht aus Berlin.` |
