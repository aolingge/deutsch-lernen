# B1 独立使用路线

B1 的目标是能在常见生活、学习、工作场景中独立表达需求、观点和理由。本仓库已有完整 B1 系统，这页只做入口。

## 能力目标

- 阅读：理解邮件、通知、规则、广告和短观点。
- 听力：理解公告、对话、访谈和广播中的关键信息。
- 写作：完成私人邮件、观点帖、正式/半正式邮件。
- 口语：共同计划、主题陈述、提问回应。

## 升级到 B2 前检查

- 能写清楚 120 词左右观点文。
- 能围绕一个主题说 2-3 分钟。
- 能听出转折、态度和同义替换。
- 能解释自己的错题原因，而不是只看答案。
