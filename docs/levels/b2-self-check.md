# B2 自检训练

B2 的任务是把“能交流”升级为“能讨论复杂问题”。你需要读懂较长材料、表达立场、说明利弊，并能把语法用于段落组织。

## 听力

任务：

听一段新闻、访谈或专题视频，做结构笔记：

```text
Thema:
These:
Argument 1:
Beispiel:
Argument 2:
Gegenposition:
Schluss:
```

合格标准：

- 能听出主题、论点和例子。
- 能区分事实、评价和推测。
- 能用 5-8 句话复述。

## 阅读

任务：

读一篇新闻评论或说明文，给每段贴标签：

- Problem
- Ursache
- Folge
- Beispiel
- Vergleich
- Kritik
- Loesung

合格标准：

- 能看懂段落之间的逻辑。
- 能找出作者态度。
- 能提取 5 个可复用表达。

## 写作

任务：写 200 词左右观点文。

结构：

```text
Einleitung: Thema und Relevanz
Position A: Vorteil oder Argument
Position B: Nachteil oder Einwand
Eigene Meinung: Abwaegung
Schluss: Empfehlung oder Ausblick
```

可用表达：

```text
Ein wichtiger Grund dafuer ist, dass ...
Demgegenueber muss man beachten, dass ...
Aus meiner Sicht ueberwiegen ...
```

合格标准：

- 每段只有一个中心功能。
- 至少使用一个让步句和一个反驳句。
- 不堆连接词，逻辑要真的成立。

## 口语

任务：就一个社会话题做 4 分钟陈述，并回答 3 个追问。

追问练习：

```text
Koennten Sie ein Beispiel nennen?
Was waere eine moegliche Loesung?
Gibt es auch Nachteile?
Wie ist die Situation in Ihrem Land?
```

合格标准：

- 能比较两个观点。
- 能用例子解释抽象词。
- 回答追问时不只说 `ja` 或 `nein`。

## 语法自检

| 项目 | 目标 |
| --- | --- |
| 被动 | 主动、过程被动、状态被动能区分 |
| 名词化 | 能把动词句改成名词结构 |
| 关系从句 | 能用来压缩信息 |
| 连接副词 | `deshalb`, `trotzdem`, `hingegen`, `ausserdem` |
| Konjunktiv II | 建议、假设、礼貌表达 |

## 下一步

如果你能稳定完成摘要、观点文和主题陈述，就进入 [C1 自检训练](c1-self-check.md)。如果目标是 DSH，开始并行使用 [DSH 备考指南](../exam/dsh-prep-guide.md)。
