# A2 自检训练

A2 的关键是能讲日常经历、安排和需求。你不只会说“我是谁”，还要能说明昨天发生了什么、接下来要做什么、遇到问题怎么处理。

## 听力

任务：

听一段日常对话，记录：

- 发生了什么问题。
- 对方建议怎么做。
- 时间、地点、价格或交通信息。

合格标准：

- 能听懂常见场景：预约、买票、问路、课程、购物、看病。
- 能识别 `aber`, `deshalb`, `dann`, `morgen`, `leider` 这类信号词。
- 能用中文或德语复述大意。

## 阅读

任务：

读一则广告、课程说明或简单规则，完成：

```text
主题：
适合谁：
需要带什么：
什么时候开始：
我是否适合：
```

合格标准：

- 能快速找出条件、时间和费用。
- 能看懂 `muss`, `kann`, `darf`, `soll` 的基本含义。
- 能判断自己下一步要报名、询问、等待还是取消。

## 写作

任务：写一封短消息，说明你不能参加约好的活动，并提出新时间。

骨架：

```text
Hallo ...,
leider kann ich am ... nicht kommen.
Ich habe ...
Koennen wir den Termin verschieben?
Am ... habe ich Zeit.
Viele Gruesse
...
```

合格标准：

- 说明取消原因。
- 提出一个新时间。
- 语气礼貌，任务完整。

## 口语

任务：讲 2 分钟“我的一天”或“上周末”。

必须包含：

- 过去发生的一件事。
- 一个时间表达。
- 一个原因句。
- 一个未来计划。

可用表达：

```text
Gestern habe ich ...
Danach bin ich ...
Ich war muede, weil ...
Naechste Woche moechte ich ...
```

## 语法自检

| 项目 | 能做到才算过 |
| --- | --- |
| Perfekt | `Ich habe gelernt.` / `Ich bin gefahren.` |
| 情态动词 | `Ich muss arbeiten.` |
| Dativ 初步 | `mit dem Bus`, `bei meiner Freundin` |
| 可分动词 | `Ich stehe um sieben Uhr auf.` |
| 基础从句 | `Ich bleibe zu Hause, weil ich krank bin.` |

## 下一步

如果你能稳定完成日常叙事和短消息，就进入 [B1 自检训练](b1-self-check.md)。A2 到 B1 的重点不是学更多单词，而是把每个场景说成“原因 + 行动 + 结果”。
