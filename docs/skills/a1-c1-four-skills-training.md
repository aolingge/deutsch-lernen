# A1-C1 四技能训练系统

四技能不是平均用力。不同等级的重点不同：A1-A2 建句子，B1 建任务，B2 建观点，C1 建学术和专业表达。

## 阅读 Lesen

| 等级 | 材料 | 训练动作 |
| --- | --- | --- |
| A1-A2 | 对话、通知、短邮件 | 找时间、地点、人物、价格 |
| B1 | 邮件、广告、规则、短观点 | 先读题，找同义替换和态度 |
| B2 | 新闻、评论、说明文 | 标段落功能和论点 |
| C1 | 长文、学术介绍、报告 | 摘要、评价论证、识别限制条件 |

## 听力 Hoeren

| 等级 | 材料 | 训练动作 |
| --- | --- | --- |
| A1-A2 | 慢速对话、课程音频 | 听数字、时间、地点 |
| B1 | 公告、对话、访谈 | 听前预测，听中抓转折 |
| B2 | 新闻、播客、较长访谈 | 做主题和细节笔记 |
| C1 | 讲座、报告、讨论 | 做结构笔记，区分论点和例子 |

## 写作 Schreiben

| 等级 | 输出 | 训练动作 |
| --- | --- | --- |
| A1-A2 | 短消息、简单邮件 | 用固定格式写清楚任务 |
| B1 | 私人邮件、观点帖、正式邮件 | 覆盖任务点，减少语序错误 |
| B2 | 观点文、投诉、申请动机 | 论点-理由-例子-限制 |
| C1 | 摘要、图表、论证文 | 引入材料，组织段落，评价观点 |

## 口语 Sprechen

| 等级 | 输出 | 训练动作 |
| --- | --- | --- |
| A1-A2 | 自我介绍、日常问答 | 固定问题反复回答 |
| B1 | 计划、陈述、提问回应 | 录音，检查是否有理由句 |
| B2 | 讨论、表达赞同反对 | 准备连接词和让步句 |
| C1 | 报告、学术讨论、追问 | 5-8 分钟报告 + 问答 |

## 每周训练配比

| 阶段 | 输入 | 输出 | 复盘 |
| --- | --- | --- | --- |
| A1-A2 | 60% | 30% | 10% |
| B1 | 45% | 40% | 15% |
| B2 | 40% | 45% | 15% |
| C1 | 35% | 50% | 15% |

## 一个可执行周循环

```text
周一：语法 + 短输出
周二：阅读 + 摘句型
周三：听力 + 跟读
周四：写作 + 改写
周五：口语录音
周六：考试或长材料专项
周日：错题复盘和下周计划
```
