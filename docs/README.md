## 学习闭环

```text
1. 选等级：A1/A2/B1/B2/C1
2. 选主线：课程、考试、赴德准备
3. 每天训练：输入 -> 语法/词汇 -> 输出 -> 复盘
4. 每周自检：听、读、写、说各做一次
5. 升级或补弱项
```

## 不要这样用

- 不要一次打开几十个资源。
- 不要只看语法解释，不写句子。
- 不要只刷阅读听力，不录口语。
- 不要把来源不明真题包当主线。
- 不要把个人材料、截图、下载资料推到公开仓库。
