# 介词和动词介词搭配

介词决定格，也决定句子的场景关系。B1 最重要的不是背中文意思，而是知道它后面常接哪一格、表示位置还是方向、和哪些动词固定搭配。

`Wechselpraeposition = 双向介词 / 表位置时常接 Dativ，表方向时常接 Akkusativ / in, an, auf 等属于这一类。`

## 固定接 Akkusativ 的介词

常见口诀可以帮助入门，但最后要回到例句。

| 介词 | 常见意思 | 例句 |
| --- | --- | --- |
| durch | 穿过、通过 | Wir gehen durch den Park. |
| fuer | 为了、给 | Das Geschenk ist fuer meinen Freund. |
| gegen | 反对、约在 | Ich bin gegen diese Idee. |
| ohne | 没有 | Ohne meinen Pass kann ich nicht reisen. |
| um | 围绕、在几点 | Der Kurs beginnt um acht Uhr. |

## 固定接 Dativ 的介词

| 介词 | 常见意思 | 例句 |
| --- | --- | --- |
| aus | 从...出来、来自 | Ich komme aus China. |
| bei | 在某人处、在某机构 | Ich wohne bei meiner Freundin. |
| mit | 和、用 | Ich fahre mit dem Bus. |
| nach | 去、之后 | Ich fahre nach Berlin. |
| seit | 自从 | Ich lerne seit einem Jahr Deutsch. |
| von | 从、来自 | Ich komme von der Arbeit. |
| zu | 去某处/某人那里 | Ich gehe zum Arzt. |

## 双向介词：位置 Dativ，方向 Akkusativ

常见双向介词：`in`, `an`, `auf`, `ueber`, `unter`, `vor`, `hinter`, `neben`, `zwischen`。

判断问题：

```text
Wo? -> Dativ
Wohin? -> Akkusativ
```

例句：

```text
Ich bin in der Schule.      Wo? Dativ
Ich gehe in die Schule.     Wohin? Akkusativ

Das Buch liegt auf dem Tisch.
Ich lege das Buch auf den Tisch.
```

## 时间介词

| 表达 | 用法 | 例句 |
| --- | --- | --- |
| um | 具体钟点 | um 8 Uhr |
| am | 日期、星期、一天中的某段 | am Montag, am Abend |
| im | 月份、季节、年份范围 | im Mai, im Sommer |
| seit | 从过去到现在 | seit zwei Jahren |
| vor | 多久以前 | vor einer Woche |
| nach | 之后 | nach dem Unterricht |
| bis | 到...为止 | bis Freitag |

听力里，时间介词常决定答案。听到改约时尤其注意 `nicht um ..., sondern um ...`。

## 地点和方向介词

| 需求 | 常用表达 |
| --- | --- |
| 去城市/国家 | nach Berlin, nach Deutschland |
| 去机构/人那里 | zum Arzt, zur Schule, zu meiner Freundin |
| 在城市/国家 | in Berlin, in Deutschland |
| 来自某地 | aus China, von der Arbeit |
| 在某人处 | bei meinen Eltern |

## 动词 + 介词

这些搭配建议整句记，不要只背中文。

| 搭配 | 格 | 例句 |
| --- | --- | --- |
| warten auf | Akkusativ | Ich warte auf den Bus. |
| sich freuen auf | Akkusativ | Ich freue mich auf die Pruefung. |
| sich freuen ueber | Akkusativ | Ich freue mich ueber deine Nachricht. |
| teilnehmen an | Dativ | Ich nehme an einem Kurs teil. |
| sprechen mit | Dativ | Ich spreche mit dem Lehrer. |
| sprechen ueber | Akkusativ | Wir sprechen ueber das Thema. |
| sich interessieren fuer | Akkusativ | Ich interessiere mich fuer Musik. |
| denken an | Akkusativ | Ich denke an meine Familie. |

## 形容词 + 介词

| 搭配 | 例句 |
| --- | --- |
| interessiert an + Dativ | Ich bin an dem Kurs interessiert. |
| zufrieden mit + Dativ | Ich bin mit dem Ergebnis zufrieden. |
| wichtig fuer + Akkusativ | Deutsch ist wichtig fuer meine Arbeit. |
| verantwortlich fuer + Akkusativ | Ich bin fuer die Planung verantwortlich. |

## 训练方法

1. 每天选 5 个介词短语造句。
2. 听力错题里专门记录时间、地点、方向介词。
3. 写作里优先用熟块：`mit dem Bus`, `in meiner Stadt`, `seit einem Jahr`。
4. 动词介词搭配做成句子卡：正面写中文场景，背面写完整德语句。
