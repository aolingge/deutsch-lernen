# B1 识别型语法补充

有些语法 B1 不一定要大量输出，但阅读和听力中要认得。先做到“看懂不慌”，再决定是否主动使用。

## Plusquamperfekt

用途：过去的过去。

结构：

```text
hatte/war + Partizip II
```

例句：

```text
Nachdem ich gegessen hatte, ging ich spazieren.
```

B1 策略：阅读中看到 `hatte/war + PII`，理解为“更早发生”。

## Futur I

结构：

```text
werden + Infinitiv
```

用途：未来、推测。

```text
Ich werde morgen lernen.
Das wird sicher schwierig sein.
```

## Zustandspassiv

`Zustandspassiv = 状态被动 / 强调结果状态 / sein + Partizip II。`

对比：

```text
Die Tuer wird geschlossen. 过程：门正在被关。
Die Tuer ist geschlossen. 状态：门是关着的。
```

## Genitiv 介词

常见：

- `wegen`：因为
- `trotz`：尽管
- `waehrend`：在...期间
- `statt/anstatt`：代替

例句：

```text
Wegen des Wetters bleiben wir zu Hause.
Trotz des Regens gehen wir spazieren.
```

口语中常听到 Dativ 用法，但考试阅读里先按标准形式识别。

## ohne...zu / statt...zu

```text
Er ging nach Hause, ohne etwas zu sagen.
Statt mit dem Auto zu fahren, nehme ich den Zug.
```

## 学习策略

| 语法 | B1 目标 |
| --- | --- |
| Plusquamperfekt | 先识别时间顺序 |
| Futur I | 能理解未来/推测 |
| Zustandspassiv | 能和过程被动区分 |
| Genitiv 介词 | 阅读中识别，不强行口语输出 |
| ohne/statt zu | 写作可加分，但先别过度使用 |
