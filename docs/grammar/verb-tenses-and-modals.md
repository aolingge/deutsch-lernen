# 动词时态和情态动词

B1 阶段最常用的是现在时、现在完成时、少量过去时，以及情态动词。

## 现在时 Praesens

现在时用于日常事实、现在发生的事，也常用来表达近期计划。

规则动词以 `machen` 为例：

| 主语 | 变位 |
| --- | --- |
| ich | mache |
| du | machst |
| er / sie / es | macht |
| wir | machen |
| ihr | macht |
| sie / Sie | machen |

例句：

```text
Ich lerne Deutsch.
我学德语。

Er geht zur Schule.
他去学校。
```

## 现在完成时 Perfekt

Perfekt 在口语里非常常见，用来讲过去发生的事。

结构：

```text
haben/sein + Partizip II
```

多数及物动词用 `haben`；表示位置移动或状态变化的动词常用 `sein`。

常见动词：

| Infinitiv | Partizip II | Hilfsverb |
| --- | --- | --- |
| gehen | gegangen | sein |
| kommen | gekommen | sein |
| fahren | gefahren | sein |
| bleiben | geblieben | sein |
| sein | gewesen | sein |
| haben | gehabt | haben |
| machen | gemacht | haben |
| sehen | gesehen | haben |
| essen | gegessen | haben |
| trinken | getrunken | haben |

例句：

```text
Ich habe Deutsch gelernt.
我学了德语。

Sie ist nach Hause gegangen.
她回家了。

Wir haben einen Film gesehen.
我们看了一部电影。
```

## 过去时 Praeteritum

B1 写作和阅读里常见 `sein`、`haben` 和情态动词的过去时。

| 原形 | 过去时 |
| --- | --- |
| sein | war |
| haben | hatte |
| gehen | ging |
| kommen | kam |
| koennen | konnte |
| muessen | musste |
| wollen | wollte |
| sollen | sollte |
| duerfen | durfte |
| moegen | mochte |

例句：

```text
Gestern war ich zu Hause.
昨天我在家。

Er hatte keine Zeit.
他没有时间。
```

## 情态动词 Modalverben

情态动词表达能力、义务、允许、愿望或建议。主句中，情态动词变位后放第二位，实义动词原形放句末。

| 原形 | 中文 | ich | du | er/sie/es | wir | ihr | sie/Sie |
| --- | --- | --- | --- | --- | --- | --- | --- |
| koennen | 能、可以 | kann | kannst | kann | koennen | koennt | koennen |
| muessen | 必须 | muss | musst | muss | muessen | muesst | muessen |
| wollen | 想要 | will | willst | will | wollen | wollt | wollen |
| sollen | 应该、被要求 | soll | sollst | soll | sollen | sollt | sollen |
| duerfen | 被允许 | darf | darfst | darf | duerfen | duerft | duerfen |
| moegen | 喜欢 | mag | magst | mag | moegen | moegt | moegen |

例句：

```text
Ich kann Deutsch sprechen.
我会说德语。

Du musst heute lernen.
你今天必须学习。

Darf ich hier sitzen?
我可以坐这里吗？
```

## 被动态 Passiv

B1 先掌握过程被动态：重点是事情正在被处理，而不是谁处理。

结构：

```text
werden + Partizip II
```

例句：

```text
Das Haus wird gebaut.
这栋房子正在被建造。

Der Brief wurde gestern geschickt.
这封信昨天被寄出了。
```

## 反身动词

反身动词表示动作回到主语自己身上。

| 主语 | Akkusativ | Dativ |
| --- | --- | --- |
| ich | mich | mir |
| du | dich | dir |
| er / sie / es | sich | sich |
| wir | uns | uns |
| ihr | euch | euch |
| sie / Sie | sich | sich |

常见搭配：

- sich freuen auf + Akkusativ：期待
- sich interessieren fuer + Akkusativ：对……感兴趣
- sich beeilen：赶快
- sich entschuldigen fuer + Akkusativ：为……道歉
- sich treffen：见面
- sich vorstellen：自我介绍

例句：

```text
Ich freue mich auf das Wochenende.
我期待周末。

Er interessiert sich fuer Musik.
他对音乐感兴趣。
```
