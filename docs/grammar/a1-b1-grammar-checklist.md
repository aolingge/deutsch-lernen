# A1-B1 全语法点清单

这页是查漏表，不是一次性背诵表。每个语法点都要完成四步：能在阅读/听力里认出，能用中文说清用途，能写 3 个自己的例句，能放进写作或口语任务。

`DaF = Deutsch als Fremdsprache / 德语作为外语 / 本页按外语学习者从 A1 到 B1 的路线组织。`

## 句子底盘

| 语法点 | B1 要求 | 学习动作 |
| --- | --- | --- |
| 陈述句 V2 语序 | 变位动词第二位 | 用 `Heute... / Deshalb... / Am Wochenende...` 各造 1 句 |
| Ja/Nein 问句 | 动词第一位 | 把 5 个陈述句改成问句 |
| W-Frage | 疑问词第一位，动词第二位 | 用 wann/wo/warum/wie 各问 1 句 |
| 句首状语倒装 | 句首不是主语时，动词仍第二位 | 写时间、地点、原因开头的句子 |
| Satzklammer 句框 | 情态、完成时、可分动词形成句框 | 标出变位动词和句末动词 |
| TeKaMoLo | 时间、原因、方式、地点的常见顺序 | 改写一个长句，调整信息顺序 |

`Satzklammer = 句框 / 变位动词和句末成分把句子夹起来 / 德语长句理解的核心。`

## 动词系统

| 语法点 | B1 要求 | 学习动作 |
| --- | --- | --- |
| 规则动词现在时 | 按人称变位 | 每天选 5 个动词造句 |
| sein/haben/werden | 高频核心动词 | 写身份、拥有、变化/被动句 |
| 强变化动词 | lesen/sieht/faehrt 等 | 把高频不规则动词放入例句 |
| 情态动词 | koennen/muessen/wollen/sollen/duerfen/moechten | 写建议、义务、许可和请求 |
| 可分动词 | 前缀回句末，完成时 ge 插入 | 练 `anrufen/einkaufen/aufstehen` |
| 不可分动词 | be-/ent-/er-/ver-/zer- 通常无 ge | 对比 `besuchen` 和 `einkaufen` |
| 反身动词 | sich + Akkusativ/Dativ | 写 `sich vorbereiten auf` 等搭配 |
| 命令式 | 请求、指示、说明 | 写 5 句生活指令 |
| 动词支配格 | helfen + Dativ, fragen + Akkusativ | 做动词支配格表 |
| 动词 + 介词 | warten auf, teilnehmen an 等 | 做 Anki 句子卡 |

## 时态和语气

| 语法点 | B1 要求 | 学习动作 |
| --- | --- | --- |
| Praesens | 事实、习惯、计划 | 讲今天安排 |
| Perfekt | 讲过去经历，haben/sein + Partizip II | 每天讲昨天发生的 5 件事 |
| Praeteritum | war/hatte/情态动词过去式 | 写过去状态和义务 |
| Plusquamperfekt | 过去的过去，B1 先识别 | 阅读中遇到先标出来 |
| Futur I | werden + Infinitiv，计划或推测 | 写 3 个未来计划句 |
| Konjunktiv II | 礼貌、建议、假设 | 背 `koennte/wuerde/waere/haette/sollte` |
| Passiv | werden + Partizip II | 主动句改被动句 |
| Zustandspassiv | sein + Partizip II，B1 先识别 | 和过程被动做对比 |

`Konjunktiv II = 第二虚拟式 / 用于礼貌请求、建议、愿望和非现实条件 / B1 写作和口语都常见。`

## 名词、冠词、格

| 语法点 | B1 要求 | 学习动作 |
| --- | --- | --- |
| 名词性别 | der/die/das 和复数一起记 | 每个新名词写性别和复数 |
| 复数形式 | -e/-er/-en/-s/不变 | 按主题整理常见复数 |
| Nominativ | 主语 | 找句子主语 |
| Akkusativ | 直接宾语、方向 | 练 `ich sehe den Mann` |
| Dativ | 间接宾语、位置、固定介词 | 练 `mit dem Bus` |
| Genitiv | 所属，B1 先识别 | 识别 `des/eines` |
| 定冠词 | der/das/die 变格 | 和格一起背 |
| 不定冠词 | ein/eine 变格 | 写一名词三格句 |
| 否定冠词 | kein 变格 | 对比 nicht 和 kein |
| 物主冠词 | mein/dein/sein/ihr/unser | 写自己的物品和关系 |
| 指示词 | dieser/jener/solcher 基本识别 | 阅读中先识别 |
| n-Deklination | der Kunde -> den Kunden 等，先识别 | 收集常见阳性弱变化名词 |

## 形容词和副词

| 语法点 | B1 要求 | 学习动作 |
| --- | --- | --- |
| 形容词作表语 | `Das ist wichtig.` 无词尾 | 对比名词前形容词 |
| 定冠词后形容词词尾 | der gute Kurs | 背常用块 |
| 不定冠词后形容词词尾 | ein guter Kurs | 写 10 个名词短语 |
| 无冠词形容词词尾 | gutes Wetter | 先识别，逐步输出 |
| 比较级 | groesser, besser, mehr | 写观点比较句 |
| 最高级 | am besten, der beste | 写推荐和评价 |
| 副词位置 | gern, oft, vielleicht 等 | 放入句框练习 |

## 介词

| 语法点 | B1 要求 | 学习动作 |
| --- | --- | --- |
| Akkusativ 介词 | durch/fuer/gegen/ohne/um | 每个介词造 1 句 |
| Dativ 介词 | aus/bei/mit/nach/seit/von/zu | 和交通、地点、时间一起练 |
| Wechselpraepositionen | in/an/auf 等，方向 A，位置 D | 画图判断方向和位置 |
| Genitiv 介词 | wegen/trotz/waehrend 等先识别 | 写 3 个常见句 |
| 时间介词 | um/am/im/seit/vor/nach/bis | 听力专项练数字和时间 |
| 地点介词 | in/an/auf/bei/zu/nach/von | 结合地图和路线练 |
| 动词介词搭配 | warten auf, sich freuen ueber | 做固定搭配表 |
| 形容词 + 介词 | interessiert an, zufrieden mit | 放进观点句 |

## 从句和连接

| 语法点 | B1 要求 | 学习动作 |
| --- | --- | --- |
| dass 从句 | 内容、观点 | `Ich denke, dass...` |
| weil/da 从句 | 原因 | 每次观点后加一个理由 |
| wenn 从句 | 条件/重复时间 | 写计划和条件 |
| als 从句 | 过去一次时间 | 写童年或过去经历 |
| bevor/nachdem/waehrend | 时间顺序 | 写流程说明 |
| obwohl 从句 | 让步 | 写转折观点 |
| ob 从句 | 是否，间接疑问 | 写礼貌询问 |
| 间接疑问句 | wann/warum/wie + 句末动词 | 正式邮件常用 |
| 关系从句 | 补充说明名词 | 描述人、物、地点 |
| Infinitiv mit zu | 计划、尝试、重要性 | `Ich habe vor, ... zu ...` |
| um...zu/damit | 目的 | 写学习目标 |
| ohne...zu/statt...zu | 不做/代替做，先识别 | B1 后期补 |
| und/aber/oder/denn | 并列连接 | 稳定输出 |
| deshalb/trotzdem/ausserdem | 连接副词，主句语序 | 写观点文 |
| einerseits/andererseits | 观点文对比 | 写优缺点 |

## 否定、数词、功能表达

| 语法点 | B1 要求 | 学习动作 |
| --- | --- | --- |
| nicht 位置 | 否定动词、形容词、句子成分 | 做句子改否定 |
| kein 变格 | 否定名词 | 和冠词表一起练 |
| nie/nichts/niemand | 否定词 | 写经历和规则 |
| 数字、日期、时间 | 听力高频 | 每天听写 5 个数字/时间 |
| 程度副词 | sehr/ziemlich/besonders/kaum | 修饰观点句 |
| 礼貌请求 | koennten/wuerden/ich haette gern | 正式邮件必用 |
| 表观点 | ich finde/denke/meine, dass | 写作口语必用 |
| 表建议 | du solltest/wir koennten/ich schlage vor | 口语 Teil 1 必用 |

## 推荐顺序

1. 先稳句子骨架、现在时、Perfekt、情态动词。
2. 再攻四格、冠词、介词、形容词词尾。
3. 然后把从句、连接词、Infinitiv mit zu 用进写作口语。
4. 最后补被动、Konjunktiv II、关系从句、固定搭配。
