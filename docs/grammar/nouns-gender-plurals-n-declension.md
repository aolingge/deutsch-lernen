# 名词性别、复数和 n-Deklination

B1 阶段名词不要只背中文意思，要一起背 `性别 + 复数 + 常用搭配`。性别决定冠词，复数影响听读判断，`n-Deklination` 会在阅读里改变词形。

`n-Deklination = 弱变化阳性名词 / 除第一格单数外常加 -n 或 -en / 常见于 der Kunde, der Student, der Mensch。`

## 每个名词都这样记

```text
der Termin, -e
einen Termin vereinbaren
Ich moechte einen Termin vereinbaren.
```

## 性别判断线索

| 常见线索 | 多数性别 | 例子 |
| --- | --- | --- |
| -ung, -heit, -keit, -schaft, -tion | die | die Anmeldung, die Gesundheit |
| -chen, -lein, -ment, -um | das | das Maedchen, das Zentrum |
| -er 表职业/人，-ling, -ismus | der | der Lehrer, der Fruehling |

线索不是百分百规则，最终以词典为准。

## 复数类型

| 类型 | 例子 | 训练方式 |
| --- | --- | --- |
| -e | der Tag, die Tage | 高频阳性/中性 |
| -er + 变音 | das Kind, die Kinder | 常见中性 |
| -en/-n | die Frage, die Fragen | 阴性很常见 |
| -s | das Auto, die Autos | 外来词常见 |
| 不变 | der Lehrer, die Lehrer | -er 结尾常见 |

## n-Deklination 常见词

| 第一格 | 非第一格例子 | 中文 |
| --- | --- | --- |
| der Kunde | Ich helfe dem Kunden. | 顾客 |
| der Student | Ich kenne den Studenten. | 大学生 |
| der Mensch | Ich sehe den Menschen. | 人 |
| der Kollege | Ich spreche mit dem Kollegen. | 同事 |
| der Name | Ich kenne seinen Namen. | 名字 |

## 每日 10 词训练

1. 写 der/die/das + 复数。
2. 写一个动词搭配。
3. 写一个 B1 句子。
4. 第二天用其中 3 个词口头复述。
