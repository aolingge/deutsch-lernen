# 比较级、最高级和程度表达

比较级和最高级在 B1 写作口语里很有用，尤其是表达优缺点、选择理由、城市/工作/生活方式比较。

## 比较级

结构：

```text
形容词 + -er + als
```

例句：

```text
Der Zug ist schneller als der Bus.
Diese Wohnung ist guenstiger als die andere.
Deutsch ist fuer mich schwieriger als Englisch.
```

## 同级比较

结构：

```text
so ... wie
```

例句：

```text
Das Fahrrad ist nicht so bequem wie das Auto.
Der Kurs ist so interessant wie der letzte Kurs.
```

## 最高级

常见结构：

```text
am + 形容词-sten
der/die/das + 形容词词尾
```

例句：

```text
Am besten lerne ich am Morgen.
Das ist die beste Loesung.
Der wichtigste Grund ist die Gesundheit.
```

## 高频不规则

| 原级 | 比较级 | 最高级 |
| --- | --- | --- |
| gut | besser | am besten |
| viel | mehr | am meisten |
| gern | lieber | am liebsten |
| hoch | hoeher | am hoechsten |
| nah | naeher | am naechsten |

## 输出模板

```text
Ich finde ..., weil es billiger und praktischer ist.
Ein Vorteil ist, dass ... viel schneller ist.
Am wichtigsten ist fuer mich, dass ...
```
