# A1-C1 语法升级路线

语法学习不要按“术语难度”排序，而要按“表达任务”排序。A1 先能说完整句，B1 能解释理由，B2-C1 能处理抽象逻辑和学术表达。

## A1：句子成形

必须掌握：

- 主句 V2 语序。
- 现在时动词变位。
- sein/haben。
- 定冠词、不定冠词基础。
- Nominativ/Akkusativ 初步。
- Ja/Nein 问句和 W-Frage。
- nicht/kein 基础。

输出目标：

```text
Ich heisse ...
Ich komme aus ...
Ich lerne Deutsch, weil ...
```

## A2：日常叙事

必须掌握：

- Perfekt 讲过去。
- 情态动词。
- Dativ 初步。
- 物主冠词。
- 可分动词。
- 时间、地点、方向介词。
- dass/weil/wenn 基础从句。

输出目标：

```text
Gestern habe ich ...
Ich muss ...
Wenn ich Zeit habe, ...
```

## B1：任务型表达

必须掌握：

- 四格、冠词、代词、形容词词尾。
- 从句和连接词：dass, weil, wenn, obwohl, ob。
- Infinitiv mit zu, um ... zu, damit。
- 关系从句基础。
- 被动和 Konjunktiv II 基础。
- 动词介词搭配。

入口：[A1-B1 全语法点清单](a1-b1-grammar-checklist.md)。

## B2：复杂观点

重点：

- 关系从句更稳定。
- 被动、状态被动、替代表达。
- 名词化结构：`die Verbesserung`, `die Entwicklung`, `die Entscheidung`。
- Partizipialattribute 先识别，再少量输出。
- 连接副词和让步/转折结构。
- Konjunktiv I 识别间接引语。

输出目标：

```text
Ein wichtiger Grund dafuer ist, dass ...
Obwohl es Nachteile gibt, ueberwiegen die Vorteile.
Die Entwicklung zeigt, dass ...
```

## C1：学术表达

重点：

- 学术名词化和功能动词结构。
- 复杂连接：waehrend, wohingegen, sofern, falls, indem。
- 被动替代表达：`sich lassen`, `man`, `sein zu + Infinitiv`。
- 引用和转述：Konjunktiv I / indirekte Rede 识别与基础使用。
- 图表和论证语言。

输出目标：

```text
Aus der Grafik geht hervor, dass ...
Der Autor weist darauf hin, dass ...
Diese Entwicklung laesst sich dadurch erklaeren, dass ...
```

## 学习方法

- A1-A2：用短句练，不追求复杂。
- B1：每个语法点绑定考试任务。
- B2：把语法用于观点和段落。
- C1：把语法用于摘要、论证和学术表达。

## 错误优先级

| 错误 | 优先级 |
| --- | --- |
| 动词位置错到影响理解 | 最高 |
| 四格和冠词高频错误 | 高 |
| 从句句末动词遗漏 | 高 |
| 形容词词尾小错 | 中 |
| 高级结构不用 | 低，先保证清楚 |
