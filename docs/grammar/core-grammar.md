## 语序

德语语序最重要的规则之一：变位动词的位置很固定。

主句中，变位动词通常在第二位：

```text
Ich lerne jeden Tag Deutsch.
我每天学德语。
```

从句中，变位动词通常放到句尾：

```text
Ich lerne Deutsch, weil ich in Deutschland leben moechte.
我学德语，因为我想在德国生活。
```

## Perfekt 现在完成时

Perfekt 常用于口语中表达过去发生的事。

基本结构：

```text
haben/sein + Partizip II
```

例句：

```text
Ich habe gestern einen Brief geschrieben.
我昨天写了一封信。

Ich bin nach Berlin gefahren.
我去了柏林。
```

## Modalverben 情态动词

常见情态动词：

- koennen：能够
- muessen：必须
- duerfen：被允许
- sollen：应该、被要求
- wollen：想要
- moechten：想要，语气更礼貌

例句：

```text
Ich muss morgen zum Arzt gehen.
我明天必须去看医生。
```

## Konjunktiv II 第二虚拟式

常用于礼貌请求、建议和假设。

```text
Koennten Sie mir bitte helfen?
您能帮我一下吗？

Ich wuerde gern einen Termin vereinbaren.
我想预约一个时间。
```

## Passiv 被动态

当重点是“事情被做了”，而不是“谁做的”，会用被动态。

```text
Der Antrag wird bearbeitet.
申请正在被处理。
```
