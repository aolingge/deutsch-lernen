# B1 学习路线

B1 的目标不是“背完一本书”，而是能在常见生活、学习和工作场景里表达需求、理解主旨、写出结构清楚的短文。

## 阶段 1：补 A2 到 B1 的语法地基

重点：

- 主句和从句语序
- 现在完成时 Perfekt
- 情态动词
- 可分动词
- 介词和格
- 形容词词尾变化
- 第二虚拟式 Konjunktiv II
- 被动态 Passiv

建议做法：

- 每天只抓 1 个语法点。
- 每个语法点写 5 个自己的句子。
- 不只看懂，要能开口复述。

## 阶段 2：按主题积累词汇

建议主题：

- 自我介绍和家庭
- 住房和搬家
- 购物和消费
- 健康和看病
- 交通和旅行
- 学校和职业培训
- 工作和求职
- 政府办事和预约
- 德国文化、节日和社会规则

每个词条建议包含：词性、中文意思、常用搭配、例句。

## 阶段 3：四项能力同步练

- 阅读：先看问题，再定位关键词。
- 听力：先预测场景，再抓数字、时间、地点、态度。
- 写作：固定结构先熟练，再慢慢增加表达。
- 口语：先保证完整句和连接词，再追求高级表达。

## 阶段 4：考试专项

B1 备考建议每周做一次小闭环：

1. 一套阅读或听力专项。
2. 一篇写作限时练习。
3. 一个口语主题录音。
4. 复盘错题和卡住的表达。
