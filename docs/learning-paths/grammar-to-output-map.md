# 语法到输出地图

语法不是为了背术语，而是为了完成表达任务。这页把常见语法点和真实输出连接起来。

## 学习顺序

```text
看规则 -> 读例句 -> 改写句子 -> 写自己的句子 -> 放进邮件/口语/作文
```

如果一个语法点不能进入输出，先不要追求“讲得很全”。

## A1-A2：把句子说完整

| 语法点 | 用来做什么 | 练习入口 |
| --- | --- | --- |
| 主句 V2 | 说完整陈述句 | [语序和句框](../grammar/word-order-and-sentence-frame.md) |
| W-Frage | 问时间、地点、人物 | [核心语法](../grammar/core-grammar.md) |
| sein/haben | 自我介绍、状态、拥有 | [A1 自检](../levels/a1-self-check.md) |
| Perfekt | 讲过去经历 | [动词时态和情态动词](../grammar/verb-tenses-and-modals.md) |
| 情态动词 | 请求、义务、许可、建议 | [动词时态和情态动词](../grammar/verb-tenses-and-modals.md) |
| Dativ/Akkusativ | 说人、物、方向和对象 | [冠词、格和代词](../grammar/cases-and-pronouns.md) |

输出任务：

```text
写 5 句介绍自己。
写 5 句昨天发生的事。
写一条改约消息。
```

## 错误优先级

| 错误 | 先修原因 |
| --- | --- |
| 动词位置错 | 直接影响理解 |
| 从句句尾动词遗漏 | B1-B2 高频扣分点 |
| 四格和冠词乱 | 写作和口语都会受影响 |
| 连接词乱用 | 段落逻辑会变假 |
| 高级结构不用 | 不是初期问题，先保证清楚 |
