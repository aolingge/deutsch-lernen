# 等级升级方法：从 A1 到 C1 怎么学

很多人卡住，不是因为资源不够，而是每一级用了同一种学习方法。A1 可以靠模仿，B1 必须开始输出，B2-C1 必须处理长文本和抽象逻辑。

## A1-A2：建立可用句子

学习重点：

- 听音和跟读比语法细节更重要。
- 词汇要带冠词、复数、例句。
- 每个语法点只问一个问题：我能不能用它说自己的事？

每天动作：

- 跟读 10 分钟。
- 写 5 个短句。
- 做 10-15 分钟互动练习。
- 复习 15 张 Anki 句子卡。

不要做：

- 不要一开始硬啃大语法书。
- 不要只背单词中文意思。
- 不要跳过发音。

## B1：建立考试和生活闭环

学习重点：

- 语法要和任务绑定：从句用于解释理由，情态动词用于规则和建议。
- 四项技能必须同步，不要只刷阅读听力。
- 错题必须变成下次动作。

每周动作：

- 阅读或听力专项两次。
- 写作两篇。
- 口语录音三次。
- 模考或模块计时一次。

## B2：建立抽象表达

学习重点：

- 主题从日常生活扩展到社会、教育、工作、技术、环境、媒体。
- 句子从“我觉得”升级到“论点-理由-例子-限制条件”。
- 阅读开始关注段落结构，而不是逐词翻译。

每周动作：

- 读两篇新闻/评论，写摘要。
- 听两段访谈/报告，做笔记。
- 写一篇正反观点文。
- 做一次 3-5 分钟口头陈述。

## C1：建立学术和专业能力

学习重点：

- 能处理长文本和隐含逻辑。
- 能在听讲座时做结构化笔记。
- 能写摘要、评论、论证文和图表说明。
- 能在讨论中澄清、追问、让步和反驳。

每周动作：

- 一篇长文精读。
- 一段讲座听力笔记。
- 一篇 250-350 词写作。
- 一次 5-8 分钟报告录音。

## 升级判断

| 判断问题 | 如果做不到 |
| --- | --- |
| 我能不能不用模板说完 2 分钟？ | 回到上一阶段做口语复述 |
| 我能不能听懂材料大意和结构？ | 降低材料难度，先做笔记训练 |
| 我能不能写出有理由和例子的段落？ | 回到 B1/B2 观点文骨架 |
| 我能不能解释错因？ | 建立错题复盘，不继续盲刷 |

## 推荐节奏

```text
A1-A2：短句和场景
B1：任务和模板
B2：观点和结构
C1：摘要、论证和学术表达
```
