# 新手学习入口

如果你第一次打开这个仓库，先不要读完全部目录。用这页完成三件事：定位等级、选主线、开始今天的最小训练。

## 第一步：判断你现在在哪一级

| 你能做到什么 | 先按什么等级学 |
| --- | --- |
| 会字母和少量问候，但不能说完整自我介绍 | A1 |
| 能介绍自己，也能说简单日常，但过去经历很卡 | A2 |
| 能说日常需求，想准备 B1 或独立生活 | B1 |
| 能交流，但新闻、观点文、讨论很吃力 | B2 |
| 能看长文，目标是大学、DSH、TestDaF 或 C1 | C1 |

下一步：进入 [A1-C1 德语总路线](a1-c1-roadmap.md)，再打开对应等级页。

## 第二步：选一条主线

| 目标 | 今天从哪里开始 |

## 第三步：今天只做一个最小闭环

```text
输入：看 1 段课文或听 1 段音频。
识别：摘 5 个词、1 个语法点、1 个句型。
输出：写 5 句或录音 1 分钟。
复盘：写下 1 个下次要避免的错误。
```

## 你什么时候该升级

升级不是“看完课程”，而是能稳定输出。每级都用自检页判断：

- [A1 自检](../levels/a1-self-check.md)
- [A2 自检](../levels/a2-self-check.md)
- [B1 自检](../levels/b1-self-check.md)
- [B2 自检](../levels/b2-self-check.md)
- [C1 自检](../levels/c1-self-check.md)

连续两周能完成当前等级自检，再进入下一级。
