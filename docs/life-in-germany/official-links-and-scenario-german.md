## 选校/申请表达

```text
Ich interessiere mich fuer den Studiengang ...
Welche Zulassungsvoraussetzungen gibt es?
Welche Sprachkenntnisse muss ich nachweisen?
Kann ich mich direkt bewerben oder ueber uni-assist?
```

## 租房/登记表达

```text
Ist die Wohnung noch frei?
Wann kann ich die Wohnung besichtigen?
Koennten Sie mir die Wohnungsgeberbestaetigung ausstellen?
Ich moechte mich anmelden.
```

## 医保/看病表达

```text
Ich brauche eine Krankenversicherung fuer mein Studium.
Ich moechte einen Termin beim Arzt vereinbaren.
Ich habe Schmerzen seit ...
Koennten Sie das bitte wiederholen?
```

## 银行/合同表达

```text
Ich moechte ein Konto eroeffnen.
Welche Unterlagen brauche ich?
Wie hoch sind die monatlichen Gebuehren?
Ich moechte den Vertrag kuendigen.
```

## 求职/实习表达

```text
Ich interessiere mich fuer ein Praktikum.
Ich moechte mich fuer die Stelle bewerben.
Koennten Sie mir bitte mehr Informationen zur Aufgabe geben?
Ich habe Erfahrung mit ...
```

## 公开安全边界

- 只放官方链接和通用表达。
- 不保存个人申请材料、合同、保险单、银行文件、签证表或截图。
- 不记录中介微信、群二维码、私人联系方式。
- 涉及法律、移民、税务、医疗、财务时，只做学习导航，不给个人结论。
