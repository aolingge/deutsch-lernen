# 德国生活指南

这个目录整理德国生活中常见场景的德语表达和办事常识。内容仅作学习和信息整理，不构成法律、移民、医疗或财务建议。

Last checked: 2026-04-22

## 常见主题

- Anmeldung：住址登记
- Wohnung：租房和合同
- Krankenkasse：医疗保险
- Arzttermin：看病预约
- Bankkonto：银行账户
- Ausbildung / Studium：职业培训和大学
- Arbeit：求职和工作
- Verkehr：公共交通和火车
- Behoerde：政府部门办事
- Kultur：节日、礼貌和社会规则

## 场景模板：预约

```text
Guten Tag,
ich moechte gern einen Termin vereinbaren.
Es geht um ...
Haetten Sie naechste Woche einen freien Termin?
Vielen Dank.
```

中文意思：

```text
您好，
我想预约一个时间。
事情是……
请问您下周有空余预约吗？
非常感谢。
```

## 风险提醒

- 具体政策会变化，重要事项请以官方网页为准。
- 不要在公开仓库发布个人申请材料或办事截图。
- 涉及签证、居留、税务、医保时，建议查官方信息或咨询专业人士。
