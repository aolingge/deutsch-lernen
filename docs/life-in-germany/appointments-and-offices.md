# 预约与办事表达

德国生活中很多事情需要提前预约，例如看病、办证、咨询课程、租房看房、政府部门办事。先掌握一套稳定表达，会比临场硬想轻松很多。

## 基础预约模板

```text
Guten Tag,
ich moechte gern einen Termin vereinbaren.
Es geht um ...
Haetten Sie naechste Woche einen freien Termin?
Vielen Dank.
```

中文：

```text
您好，
我想预约一个时间。
事情是……
请问您下周有空余预约吗？
谢谢。
```

## 改约

```text
Guten Tag,
ich habe am [Datum] um [Uhrzeit] einen Termin.
Leider kann ich zu diesem Termin nicht kommen.
Waere es moeglich, den Termin zu verschieben?
```

关键词：

- `verschieben`：改期
- `ein neuer Termin`：一个新预约
- `leider`：很遗憾
- `moeglich`：可能的

## 取消预约

```text
Guten Tag,
ich muss meinen Termin am [Datum] leider absagen.
Vielen Dank fuer Ihr Verstaendnis.
```

关键词：

- `absagen`：取消
- `der Termin`：预约
- `das Verstaendnis`：理解

## 询问需要带什么

```text
Welche Unterlagen muss ich mitbringen?
Brauche ich meinen Ausweis?
Soll ich das Formular vorher ausfuellen?
```

常见材料词：

| Deutsch | 中文 |
| --- | --- |
| der Ausweis | 身份证件 |
| der Reisepass | 护照 |
| das Formular | 表格 |
| die Bescheinigung | 证明 |
| die Anmeldung | 登记 |
| die Krankenversicherungskarte | 医保卡 |

## 礼貌表达

```text
Koennten Sie mir bitte helfen?
Koennten Sie das bitte wiederholen?
Ich habe noch eine Frage.
Vielen Dank fuer Ihre Hilfe.
```

## 风险提醒

- 具体材料和政策会变化，请以对应机构官方页面为准。
- 不要把自己的预约确认邮件、申请表、证件照片或办事截图上传到公开仓库。
