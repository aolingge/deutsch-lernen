# Deutsch Lernen

Languages: [简体中文](README.md) | English

An open German-learning knowledge base for Chinese speakers. It organizes A1-C1 study paths, B1/DSH/TestDaF preparation, Germany study/life readiness, and long-term learning methods into reusable public notes.

> Status: early public scaffold. Content is being expanded, and the repository only accepts public, redacted, reusable material.

## Who It Helps

- Chinese-speaking learners moving from A1 toward C1.
- Learners preparing for Goethe, telc, OeSD, TestDaF, DSH, or DTZ.
- Students or newcomers preparing for life, study, work, and administration in Germany.
- People turning scattered German notes into a long-term learning system.

## Start Here

- New to the repository: start with the [system map](docs/README.md) and [start-here guide](docs/learning-paths/start-here.md).
- Planning A1-C1: read the [A1-C1 roadmap](docs/learning-paths/a1-c1-roadmap.md), then use the level pages and self-checks.
- Starting from zero: use the [zero-to-B1 30-day starter](docs/learning-paths/zero-to-b1-30-day-starter.md) to build a daily loop.
- Learning grammar but not using it yet: start with the [grammar-to-output map](docs/learning-paths/grammar-to-output-map.md).
- Studying B1 systematically: use the [B1 roadmap](docs/learning-paths/b1-roadmap.md), then pair [core grammar](docs/grammar/core-grammar.md) with [theme vocabulary](docs/vocabulary/theme-vocabulary.md).
- Preparing for exams: start with the [exam decision tree](docs/exam/exam-decision-tree.md) and [exam map](docs/exam/german-exam-map-a1-c1-dsh.md), then choose the relevant exam guide.
- Preparing for Germany: see [Germany Prep](docs/germany-prep/README.md), the [study and life checklist](docs/life-in-germany/study-and-life-prep-checklist.md), and [appointments/offices phrases](docs/life-in-germany/appointments-and-offices.md).
- Looking for public resources: use [resource packages](docs/resources/resource-packages.md) and [public resources](docs/resources/public-resources.md).

## Maintenance Principles

1. Practical before complete: every note should prefer examples, scenarios, and common mistakes.
2. Chinese explanations first: new concepts should be explained clearly in Mandarin before German terminology is introduced.
3. No private material: personal notes, chat logs, account details, original screenshots, or classroom/private group material do not belong here.
4. Public sources only: link official or clearly public resources instead of copying protected content.
5. Conservative automation: scheduled jobs may only process redacted reusable content.
