# Privacy Rules

这个仓库是公开仓库。任何提交到这里的内容，都默认全网可见、可复制、可被搜索引擎索引。

## 绝对不要提交

- 真实姓名、手机号、邮箱、住址、证件号、学校学号、公司信息
- 登录凭据、API key、Cookie、token、连接串
- 私人聊天记录、课堂群消息、老师或同学信息
- 原始截图、录音、视频、扫描件
- 未脱敏的 Obsidian 笔记、OneNote 导出、浏览器导出
- 受版权保护的教材全文、试题全文、课程讲义全文

## 可以提交

- 自己改写后的学习方法、语法解释、词汇表和例句
- 不包含个人经历细节的通用德国生活流程说明
- 官方公开网页链接和简短说明
- 自己创作的练习题、模板和学习计划
- 已确认可公开授权的资源

## 脱敏检查清单

提交前至少确认：

- 没有姓名、电话、邮箱、地址、账号、订单号、票据号
- 没有真实课堂、群聊、会议、文档的私有链接
- 没有复制教材或课程材料的大段原文
- 没有图片中的个人信息
- Git 历史里没有误提交的敏感文件

## 自动化任务边界

定时任务可以做：

- 补全公开内容目录
- 改写通用德语知识点
- 检查链接和 Markdown 格式
- 提醒哪些主题需要人工确认

定时任务不可以做：

- 自动上传本地原始资料
- 自动发布未脱敏的个人笔记
- 自动读取或提交密钥、Cookie、账号文件
- 自动复制受版权保护的大段内容
