# Contributing

欢迎补充德语学习内容。为了让仓库长期可维护，请遵守这些规则。

## 内容格式

- 文件使用 Markdown。
- 标题尽量清楚，例如 `## weil / dass 从句语序`。
- 每个语法点建议包含：中文解释、德语例句、常见错误、练习提示。
- 每个生活主题建议包含：适用场景、关键词、常见步骤、风险提醒、官方链接。

## 写作风格

- 优先写给中文母语学习者。
- 新术语第一次出现时说明它是做什么的。
- 德语例句尽量配中文解释。
- 不追求一次写完，允许短小但准确的条目。

## 提交前检查

```powershell
pwsh ./tools/privacy-scan.ps1 .
```

如果脚本提示疑似隐私或密钥，请先人工确认并删改。
